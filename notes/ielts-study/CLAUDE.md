## ⚠️ 工作规则（必须遵守）

### 复盘记录规则
1. **统一记在一个文件**：`06_复盘/进度记录.md`
   - 每天追加一个 `## Day X - YYYY-MM-DD` 章节
   - 不再创建 `日复盘/` 子目录
2. **截图命名格式**：`YYYY-MM-DD-任务名.png`（用日期开头）
   - ✅ `2026-06-14-语料库.png`
   - ❌ `01-语料库.png`（不知道哪天）
3. **生词本**：平时写到**纸质小单词本**（淘宝 5 元的口袋本）
   - 中午 + 晚上各翻 15 分钟
   - 周末可以录到 Obsidian 错词本备份

- 每天学习完成后，`git add .` → `git commit -m "DayX: 完成内容"` → `git push`
- 远程仓库：`swording-k/notes`（monorepo），ielts-study 在 `notes/ielts-study/` 子目录
- 保持 GitHub 仓库同步

### 学习计划更新
- 用户调整计划后，立即更新 `学习计划.md`
- 用户讨论出的新规则，立即更新 `CLAUDE.md`

## 🎤 口语学习流程

### 麦门讲义使用
- PDF位置：`04_口语/素材/麦门雅思独家——2026年1~4月口语合集-大讲义.pdf`
- 题库文本：`04_口语/素材/麦门讲义内容.txt`

### 学习流程
1. **浏览题目** - 先看题目，不看答案，自己试着说
2. **查看答案** - 学习参考答案的表达逻辑
3. **跟读模仿** - 跟读答案，提升语感
4. **自己回答** - 用学到的方式重新回答，录音
5. **提交分析** - 把录音转文字，提交给 Claude 分析

### 用户口语练习笔记
- 位置：`04_口语/练习/YYYY-MM-DD-话题名.md`
- 格式：题目 → 用户回答 → 参考答案 → Claude 分析

### 任务分配
- 用户：看视频→自己回答→语音输入到笔记
- Claude：根据参考答案分析用户回答，给出改进建议
