## ⚠️ 工作规则（必须遵守）

### 复盘记录规则
1. **每天写复盘**：`06_复盘/日复盘/YYYY-MM-DD-DayX.md`
2. **截图必须嵌入对应任务**：
   - 截图放在 `attachments/` 目录
   - 命名格式：`01-语料库.png`、`02-词汇真经.png`、`03-179听力.png`、`04-538阅读.png`、`05-口语薛麦.png`
   - 每个任务的截图要紧跟在任务描述后面，不要把所有截图堆在一起
   - 格式：任务标题 → 进度说明 → 截图

3. **复盘格式**：
   ```
   ## ✅ 今日完成

   ### 词汇真经
   - **进度**：xxx
   - **正确率**：xxx
   - **截图**：![描述|300](attachments/02-词汇真经.png)
   ```

### Git 同步规则
- 每天学习完成后，`git add .` → `git commit -m "DayX: 完成内容"` → `git push`
- 保持 GitHub 仓库同步

### 学习计划更新
- 用户调整计划后，立即更新 `学习计划.md`
- 用户讨论出的新规则，立即更新 `CLAUDE.md`

## 🎤 口语学习流程

### 麦门讲义使用
- PDF位置：`04_口语/素材/麦门雅思独家——2026年1~4月口语合集-大讲义.pdf`
- 题库文本：`04_口语/素材/麦门讲义内容.txt`

### 学习流程
1. **浏览题目** - 先看题目，不看答案，自己试着说
2. **查看答案** - 学习参考答案的表达逻辑
3. **跟读模仿** - 跟读答案，提升语感
4. **自己回答** - 用学到的方式重新回答，录音
5. **提交分析** - 把录音转文字，提交给 Claude 分析

### 用户口语练习笔记
- 位置：`04_口语/练习/YYYY-MM-DD-话题名.md`
- 格式：题目 → 用户回答 → 参考答案 → Claude 分析

### 任务分配
- 用户：看视频→自己回答→语音输入到笔记
- Claude：根据参考答案分析用户回答，给出改进建议
