| **写不对** | 拼写错误 | 王陆语料库 |

> S1/S3 是送分题，场景固定（旅游、银行、医院、图书馆、论文）

### 阅读

| 题型 | 策略 |
|------|------|
| 单词题/句子题/段落题/匹配题 | 平行阅读法 |

### 写作

| 分数 | 含义 |
|------|------|
| 6分 | 及格 |
| 7分 | 语言地道 |
| 8-9分 | 有思想价值 |

> 四维评分：TA / CC / LR / GRA

### 口语

| Part | 时长 | 关键 |
|------|------|------|
| Part 1 | 5min | 日常话题，自然放松 |
| Part 2 | 1+2min | Cue Card，串题故事 |
| Part 3 | 5min | 深入讨论，逻辑沟通 |

> 关键词：背诵输入、句型为重、自然放松、逻辑沟通

## 📊 三阶段计划

### 第一阶段（第1-4周）：打基础

**目标**：词汇量积累 + 方法论入门

**语料库节奏**：
- 每天 2 个 test
- 新 test 目标 70%
- 第二天先刷昨天错词到 80%+，再刷新 test

| 时间  | 任务                  | 形式  | 说明                       |
| --- | ------------------- | --- | ------------------------ |
| 早上  | 语料库 2test + 复习错词    | 听写  | 先刷昨天错词→80%+，再刷新test→70%+ |
| 下午  | 听力 S1+S2 + 阅读 P1+P2 | 做真题 | 先学平行阅读法                  |
| 晚上  | 口语 Part1 跟练         | 跟练  | 薛麦视频 1h                  |
| 空闲时 | 刘洪波 179             | 听   | 训练听到→反应词义                |
| 空闲时 | 刘洪波 538             | 看   | 训练同义替换识别                 |
| 空闲时 | 词汇真经 一组             | 看/记 | 背拼写                      |

**本周任务**：
- [ ] 语料库 Chapter 3-5，正确率目标 60%
- [ ] 词汇真经过完第1遍（所有单词见一面）
- [ ] 179 听音频过完第1遍（听到能反应）
- [ ] 538 考点词过完第1遍（看到能识别替换）
- [ ] 口语 Part1 所有话题过一遍
- [ ] 剑雅 7 完成

---

### 第二阶段（第5-8周）：强化

**目标**：方法论熟练 + 素材积累

**语料库节奏**：
- 每天 2 个 test
- 继续"错词复习→80%+，刷新test"的节奏

| 时间 | 任务 | 形式 | 说明 |
|------|------|------|------|
| 早上 | 语料库 2test + 复习错词 | 听写 | 先刷昨天错词→80%+，再刷新test |
| 上午 | 词汇真经 50个 + 538 复盘 | 看 | 重点复习高频替换 |
| 下午 | 听力 S1-S4 + 阅读 P1-P3 | 做真题 | 剑雅 14-15 |
| 晚上 | 写作框架学习 | 看课 | Simon/杜仕明 |
| 晚上 | 口语 Part2 串题素材 | 整理 | 薛麦视频 + 自己故事 |
| 空闲时 | 179 复习错词 | 听 | 重点听反应不出来的词 |

**本周任务**：
- [ ] 语料库正确率提升到 80%
- [ ] 剑雅 14-15 完成
- [ ] 写作框架课看完
- [ ] 口语 Part2 素材整理完（串题故事）

### 第三阶段（第9-12周）：冲刺

**目标**：套题训练 + 状态保持

**语料库节奏**：
- 每天 2 个 test
- 目标 95% 正确率

| 时间 | 任务 | 形式 | 说明 |
|------|------|------|------|
| 早上 | 语料库 2test + 复习错词 | 听写 | 目标95%正确率 |

| 空闲时 | 179 快速过 | 听 | 保持耳朵敏感度 |

**本周任务**：
- [ ] 语料库正确率目标 95%
- [ ] 剑雅 16-20 完成
- [ ] 全科模考至少 3 次
- [ ] 写作每天写一篇并批改
