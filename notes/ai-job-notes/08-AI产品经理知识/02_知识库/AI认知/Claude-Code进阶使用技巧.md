# Claude Code 进阶使用技巧

> 来源：抖音视频 | 2026-04-13

## 六个进阶阶段

Claude Code 的使用分为六个阶段，每个阶段都有核心技能与避坑要点：

### 1. 提示工程师
- **核心**：撰写清晰提示词，避免单向命令导致的平庸产出
- **关键**：学会判断 AI 生成内容质量，掌握基础终端知识

### 2. 规划者
- **核心**：启用计划模式实现双向协作
- **关键**：通过提问引导 Claude 提供方案，避免仅依赖文字指令

### 3. 上下文工程师
- **核心**：上下文窗口管理
- **关键技能**：
  - 使用 `/clear` 指令重置窗口
  - 手动总结避免上下文腐化
  - 精准选择必要上下文，避免信息过载

### 4. 外部能力使用者
- **核心**：根据项目选择合适的 MCP 工具和框架
- **关键**：理解技术模块协同原理，避免盲目安装插件

### 5. 技能使用者
- **核心**：定制专属技能提升效率
- **关键**：利用技能创建器优化现有技能，可按项目配置

### 6. 云代码管理者
- **核心**：通过多会话、工作树、子代理或代理团队实现规模化协作
- **注意**：代理团队为实验性功能，需通过特定指令启用；多终端协作存在收益递减规律

## 核心要点

> [!摘要]
> Claude Code 的进阶路径是从"使用者"到"管理者"：从基础的提示词撰写，到上下文管理，到工具集成，到技能定制，最后到多代理协作。
