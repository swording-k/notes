# 深度研究方法：横纵分析法

> 来源：抖音-数字生命卡兹克 | 2026-04-13

## 什么是横纵分析法

一套用AI进行深度研究的通用框架，核心是两条轴：

### 纵向分析（时间轴）
沿时间线，把一个东西从诞生到现在的完整故事还原出来：
- 它是怎么来的？谁做的？
- 中间经历了什么？
- 为什么在某个节点突然爆发/掉头？

### 横向分析（切面轴）
在当下时间点，与同赛道的其他东西放在一起对比：
- 跟竞品比有什么不同？
- 用户为什么选它不选别的？
- 它在整个赛道里是什么位置？

## 核心价值

**两条轴交叉看**：纵向告诉你它是怎么走到今天的，横向告诉你它今天站在哪。

单独看任何一条轴都看不到的东西：
- 今天的某个优势，是三年前一个不起眼的决策慢慢积累出来的
- 今天的某个短板，是当初一个合理的选择变成了包袱

## 适用范围

| 类型 | 例子 |
|------|------|
| 产品/工具 | Hermes Agent、Cursor、Claude Code |
| 公司/组织 | Anthropic、字节跳动、OpenAI |
| 技术概念 | MCP协议、RAG、Agent框架 |
| 人物 | 行业关键人物职业轨迹 |

## 使用方法

1. 把Prompt中的「研究对象」替换为想研究的内容
2. 发送给支持Deep Research的AI（ChatGPT、Claude、豆包等）
3. AI会自动进行横纵分析，生成1-3万字研究报告

## 提示工程能力提升

- 模型选型：结合业务场景选择，而非仅看技术指标
- 模型微调：重点讲数据收集处理、输出标准定义及迭代思路
- 评测体系：建立真实场景的测试标准，关注准确率与幻觉比例
- Agent搭建：说明如何保证工作流效率与稳定性，如加入反思检查节点

> [!摘要]
> 横向追时间深度，横向追同期广度，最后交汇出判断。AI让研究效率大幅提升，但真正的方向判断仍需人来定。
