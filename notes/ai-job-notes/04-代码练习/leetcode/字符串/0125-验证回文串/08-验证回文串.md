# LeetCode 125. 验证回文串

## 题目链接
https://leetcode.cn/problems/valid-palindrome/

## 题目描述
给定一个字符串，判断它是否是回文串。
- 转换规则：所有大写字母转小写，忽略非字母数字字符
- 回文串：正着读和反着读是一样的

## 示例
```
示例 1：
输入："A man, a plan, a canal: Panama"
输出：true
解释：转换为 "amanaplanacanalpanama" 后是回文

示例 2：
输入："race a car"
输出：false
解释：转换为 "raceacar" 后不是回文

示例 3：
输入：" "
输出：true
解释：空字符串是回文
```

## 约束条件
- 1 ≤ s.length ≤ 2 × 10⁵
- s 仅包含可打印的 ASCII 字符

## 解题思路

### 方法：双指针
1. 先清理字符串：转小写，只保留字母数字
2. 双指针从两端向中间比较

### 代码实现
```python
import re

class Solution:
    def isPalindrome(self, s: str) -> bool:
        # 清理：只保留字母数字，转小写
        cleaned = re.sub(r'[^a-z0-9]', '', s.lower())
        # 双指针验证
        left, right = 0, len(cleaned) - 1
        while left < right:
            if cleaned[left] != cleaned[right]:
                return False
            left += 1
            right -= 1
        return True
```

### 不使用正则的版本
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left, right = 0, len(s) - 1
        while left < right:
            # 跳过非字母数字字符
            while left < right and not s[left].isalnum():
                left += 1
            while left < right and not s[right].isalnum():
                right -= 1
            # 转小写比较
            if s[left].lower() != s[right].lower():
                return False
            left += 1
            right -= 1
        return True
```

## 复杂度分析
- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
