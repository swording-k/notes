# LeetCode 844. 比较含退格的字符串

## 题目链接
https://leetcode.cn/problems/backspace-string-compare/

## 题目描述
给定两个字符串 s 和 t，用 `#` 表示退格键。
比较两个字符串是否相等。

## 示例
```
示例 1：
输入：s = "ab#c", t = "ad#c"
输出：true
解释：s 变成 "ac"，t 也变成 "ac"

示例 2：
输入：s = "ab##", t = "c#d#"
输出：true
解释：s 变成 ""，t 也变成 ""

示例 3：
输入：s = "a#c", t = "b"
输出：false
解释：s 变成 "c"，t 是 "b"
```

## 提示
- 1 ≤ s.length, t.length ≤ 200
- s 和 t 只包含小写字母和 '#'

## 解题思路

### 方法1：栈模拟 O(n) ✅ 推荐
```python
class Solution:
    def backspaceCompare(self, s: str, t: str) -> bool:
        def process(string):
            stack = []
            for char in string:
                if char == '#':
                    if stack:
                        stack.pop()
                else:
                    stack.append(char)
            return ''.join(stack)

        return process(s) == process(t)
```

### 方法2：双指针从后往前 O(n)
```python
class Solution:
    def backspaceCompare(self, s: str, t: str) -> bool:
        i, j = len(s) - 1, len(t) - 1
        skip_s, skip_t = 0, 0

        while i >= 0 or j >= 0:
            while i >= 0:
                if s[i] == '#':
                    skip_s += 1
                    i -= 1
                elif skip_s > 0:
                    skip_s -= 1
                    i -= 1
                else:
                    break

            while j >= 0:
                if t[j] == '#':
                    skip_t += 1
                    j -= 1
                elif skip_t > 0:
                    skip_t -= 1
                    j -= 1
                else:
                    break

            if i >= 0 and j >= 0:
                if s[i] != t[j]:
                    return False
            elif i >= 0 or j >= 0:
                return False

            i -= 1
            j -= 1

        return True
```

### 核心思想
- 用栈模拟打字机的退格效果
- 遇到普通字符入栈
- 遇到 '#' 弹栈（模拟退格）
- 最后比较两个栈的内容
