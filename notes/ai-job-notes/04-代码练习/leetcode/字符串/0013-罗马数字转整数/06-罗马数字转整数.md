# LeetCode 13. 罗马数字转整数

## 题目链接
https://leetcode.cn/problems/roman-to-integer/

## 题目描述
将罗马数字转换为整数。罗马数字包含七种符号：
- I(1)、V(5)、X(10)、L(50)、C(100)、D(500)、M(1000)

通常从左到右大数在前，但存在六种减法情况：
- I 在 V、X 前表示 4 和 9
- X 在 L、C 前表示 40 和 90
- C 在 D、M 前表示 400 和 900

## 示例
```
示例 1：
输入: "III"
输出: 3

示例 2：
输入: "LVIII"
输出: 58

示例 3：
输入: "MCMXCIV"
输出: 1994
解释：M=1000, CM=900, XC=90, IV=4
```

## 约束条件
- 1 ≤ s.length ≤ 15
- 仅包含字符：'I', 'V', 'X', 'L', 'C', 'D', 'M'
- 表示整数范围：[1, 3999]
- 输入是有效的罗马数字

## 解题思路

### 方法：哈希表 + 减法规则
1. 建立罗马数字到整数的映射表
2. 从左到右遍历
3. 如果当前字符代表的值小于下一个字符的值，就减去当前值
4. 否则，加上当前值

### 代码实现

```python
class Solution:
    def romanToInt(self, s: str) -> int:
        roman_map = {
            'I': 1, 'V': 5, 'X': 10,
            'L': 50, 'C': 100, 'D': 500, 'M': 1000
        }
        result = 0
        for i in range(len(s)):
            if i + 1 < len(s) and roman_map[s[i]] < roman_map[s[i + 1]]:
                result -= roman_map[s[i]]
            else:
                result += roman_map[s[i]]
        return result
```

## 复杂度分析
- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
