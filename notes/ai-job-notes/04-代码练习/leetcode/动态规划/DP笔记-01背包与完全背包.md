# 动态规划笔记：0-1背包与完全背包

> 本笔记总结动态规划的核心概念，重点区分 0-1 背包和完全背包

---

## 一、动态规划核心步骤

### 1. 状态定义
```
dp[i] = 考虑前 i 个物品（或容量为 i）时的最优解
```

### 2. 转移方程
```
dp[i] = min(dp[i-1], dp[i-weight] + cost)  # 选或不选
```

### 3. 初始化
```
dp[0] = 0        # 容量为0，最优代价是0
dp[others] = inf # 或 amount+1
```

### 4. 遍历顺序
```
根据问题类型决定（见下文）
```

---

## 二、0-1 背包 vs 完全背包

| 类型 | 物品数量 | 循环顺序 | 代码结构 |
|------|---------|---------|---------|
| **0-1 背包** | 每个物品只能选一次 | 从大到小 | 用 `dp_new` 或从大到小 |
| **完全背包** | 每种物品无限选 | 从小到大 | 直接遍历 |

---

## 三、完全背包（物品无限）

### 适用场景
- 零钱兑换（硬币无限）
- 完全平方数（平方数无限）

### 代码模板

```python
# 完全背包：从小到大遍历
for i in range(amount + 1):        # 先循环容量
    for coin in coins:              # 再循环物品
        if i >= coin:
            dp[i] = min(dp[i], dp[i - coin] + 1)
```

### 原因
```
dp[i-coin] < i，已经计算过了
同一物品可以用多次，所以从小到大没问题
```

## 四、0-1 背包（物品唯一）

### 适用场景
- 华为 P4625（每个张量只能选一次）
- 分割等和子集（每个数只能用一次）

### 代码模板（两种写法）

#### 写法1：用新数组 dp_new（推荐新手）

```python
dp = [inf] * (max_space + 1)
dp[0] = 0

for i in range(n):                 # 先循环物品
    dp_new = dp[:]                  # ✅ 复制数组

    for j in range(max_space - weight[i] + 1):  # 从小到大
        if dp[j] != inf:
            dp_new[j + weight[i]] = min(dp_new[j + weight[i]], dp[j] + cost)

    dp = dp_new                    # 复制回去
```

#### 写法2：直接在 dp 上更新（从大到小）

```python
dp = [inf] * (max_space + 1)
dp[0] = 0

for i in range(n):                 # 先循环物品
    for j in range(max_space, weight[i] - 1, -1):  # ✅ 从大到小
        if dp[j - weight[i]] != inf:
            dp[j] = min(dp[j], dp[j - weight[i]] + cost)
```

### 原因
```
如果从小到大：dp[j-weight] 已经被本轮更新过
导致同一物品被重复使用 ❌

如果从大到小：dp[j-weight] 还是上一轮的值
保证每个物品只被使用一次 ✅
```

## 五、图解：从小到大的问题

### 错误示范

```
物品：空间=3, 代价=5 的张量
max_space = 10

从小到大遍历（错误）：
j=3:  dp[3] = min(inf, dp[0]+5) = 5     ✅ 第一次用
j=4:  dp[4] = min(inf, dp[1]+5) = inf
j=5:  dp[5] = min(inf, dp[2]+5) = inf
j=6:  dp[6] = min(inf, dp[3]+5) = 5+5=10  ❌ 又用了张量0！
      dp[3] 是本轮更新过的值！
```

### 正确示范

```
从大到小遍历（正确）：
j=10: dp[10] = min(inf, dp[7]+5) = inf
j=9:  dp[9] = min(inf, dp[6]+5) = inf
...
j=6:  dp[6] = min(inf, dp[3]+5) = 5     ✅ dp[3]还是旧值
j=5:  dp[5] = min(inf, dp[2]+5) = inf
j=4:  dp[4] = min(inf, dp[1]+5) = inf
j=3:  dp[3] = min(inf, dp[0]+5) = 5     ✅ 第一次用
```

### P4625 华为题（从大到小）

```python
for i in range(n):
    for j in range(max_space, v[i] - 1, -1):  # 从大到小
        if dp[j - v[i]] != inf:
            dp[j] = min(dp[j], dp[j - v[i]] + cost)
```

### 什么时候用哪种？

| 方法 | 适用场景 |
|------|---------|
| `dp_new = dp[:]` | 每层有多个选项需要同时考虑 |
| 从大到小 | 只有一个物品（或多个物品但只需简单 min） |

## 七、实例：华为 P4625 题解

### 题目
```
n个张量，每个选一次（swap或recompute），空间>=m，代价最小
```

### 代码

```python
class Solution:
    def function(self, m, n, v, swap, rep):
        max_space = m + max(v)          # 开大一点的数组
        dp = [float('inf')] * (max_space + 1)
        dp[0] = 0

        for i in range(n):             # 遍历每个张量
            cost = min(swap[i], rep[i])  # 选代价小的方案
            weight = v[i]

            for j in range(max_space, weight - 1, -1):  # 从大到小
                if dp[j - weight] != float('inf'):
                    dp[j] = min(dp[j], dp[j - weight] + cost)

        result = min(dp[m:])           # 空间 >= m 的最小代价
        if result == float('inf'):
            return "error"
        return int(result)
```

## 八、实例：零钱兑换（完全背包）

### 题目
```
硬币无限，空间=amount，求最少硬币数
```

### 代码

```python
class Solution:
    def coinChange(self, coins, amount):
        dp = [amount + 1] * (amount + 1)  # 初始化为不可能的大值
        dp[0] = 0

        for i in range(1, amount + 1):   # 先循环金额（从小到大）
            for coin in coins:            # 再循环硬币
                if i >= coin:
                    dp[i] = min(dp[i], dp[i - coin] + 1)

        return -1 if dp[amount] > amount else dp[amount]
```

## 九、总结

```
1. 完全背包（无限物品）→ 从小到大遍历
2. 0-1 背包（唯一物品）→ 从大到小遍历 或 用 dp_new
3. 关键：保证每个物品只被使用一次
```
