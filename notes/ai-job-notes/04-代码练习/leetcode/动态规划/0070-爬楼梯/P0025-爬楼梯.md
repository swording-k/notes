# 0070. 爬楼梯 / Climbing Stairs

> 原题链接：https://leetcode.cn/problems/climbing-stairs/

## 题目描述

假设你正在爬楼梯，需要n阶才能到达楼顶。每次你可以爬1或2个台阶。求有多少种不同的方法可以爬到楼顶。

## 输入格式

一个整数n（阶数）

## 输出格式

整数，表示爬到楼顶的不同方法数

## 示例

输入：
```
2
```

输出：
```
2
```

说明：有两种方法爬到楼顶：1阶+1阶，或2阶
