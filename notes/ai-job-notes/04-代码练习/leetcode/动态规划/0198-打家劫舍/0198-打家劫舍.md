# 0198. 打家劫舍 / House Robber

> 原题链接：https://leetcode.cn/problems/house-robber/

## 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

## 示例

```
输入: 1 2 3 1
输出: 4

说明: 偷窃1号房屋(金额=1)，然后偷窃3号房屋(金额=3)
      偷窃到的最高金额 = 1 + 3 = 4
```

---

## 问题本质

给定数组 `a1, a2, ..., an`，从中选出若干个**互不相邻**的元素使得和最大。

---

## 思路解析

### 1. 状态定义

`f(i)` = 考虑了前 i 个元素的最优解（偷到第 i 个房屋时的最大收益）

### 2. 状态转移

考虑第 i 个房屋选不选：

- **不选 ai**：问题变成从前 i-1 个元素选最优解 → `f(i-1)`
- **选 ai**：则 ai-1 不能选，问题变成从前 i-2 个元素选最优解 → `f(i-2) + ai`

```
f(i) = max(f(i-1), f(i-2) + ai)
```

### 3. 初始化

```
f(0) = a0
f(1) = max(a0, a1)
```

### 4. 最终答案

`f(n-1)` 即 `dp[-1]`

---

## 代码实现

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]

        # dp[i] 表示偷到第 i 个房屋时的最高金额
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])

        for i in range(2, n):
            dp[i] = max(dp[i-1], dp[i-2] + nums[i])

        return dp[-1]


# ================ ACM模式 ================
if __name__ == "__main__":
    import sys
    nums = list(map(int, sys.stdin.read().split()))
    solution = Solution()
    print(solution.rob(nums))
```

## 空间优化 O(1)

观察转移方程：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`

每一步只依赖前两个状态，可以用两个变量滚动优化：

```python
def rob(self, nums: list[int]) -> int:
    n = len(nums)
    if n == 0:
        return 0
    if n == 1:
        return nums[0]

    prev2 = nums[0]                    # dp[i-2]
    prev1 = max(nums[0], nums[1])     # dp[i-1]

    for i in range(2, n):
        cur = max(prev1, prev2 + nums[i])
        prev2, prev1 = prev1, cur      # 滚动

    return prev1
```

## 面试问答

**Q: 这个算法的思想是什么？**

A: 动态规划。定义 `dp[i]` 表示偷到第 i 个房屋时的最大收益。
- 如果不选当前房屋，那就是前一个房屋的最优结果 `dp[i-1]`
- 如果选当前房屋，那前一个房屋就不能选，只能加上第 `i-2` 个房屋的最优结果 `dp[i-2]`
所以每一步就是在"选当前 or 不选当前"房屋里取最大值。
每一步只依赖前两个状态，可以用两个变量滚动优化，空间做到 **O(1)**，时间复杂度 **O(n)**。
