# LeetCode 23. 电话号码的字母组合

> 原题链接：https://leetcode.cn/problems/letter-combinations-of-a-phone-number/

## 题目描述

给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。

数字到字母的映射如下（与电话按键相同）：

```
2: abc    5: jkl    8: tuv
3: def    6: mno    9: wxyz
4: ghi    7: pqrs
```

## 示例

```
示例 1：
输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]

示例 2：
输入：digits = ""
输出：[]

示例 3：
输入：digits = "2"
输出：["a","b","c"]
```

---

## 解题思路

### 方法：回溯算法

**核心思想**：每个数字对应多个字母，依次选择每个数字的每个字母

**关键点**：
- 建立数字到字母的映射表
- 按索引遍历数字字符串
- 每个位置有多个字母选择

---

## 图解回溯过程

**示例：digits = "23"**

```
                ""
      ┌─────────┼─────────┐
      │         │         │
     [a]       [b]       [c]          ← 第一层：选 '2' 的哪个字母
   ┌──┼──┐   ┌──┼──┐   ┌──┼──┐
   │  │  │   │  │  │   │  │  │
  [d][e][f] [d][e][f] [d][e][f]        ← 第二层：选 '3' 的哪个字母
   │  │  │   │  │  │   │  │  │
  ad ae af  bd be bf  cd ce cf         ← 收集结果

结果：["ad","ae","af","bd","be","bf","cd","ce","cf"]
```

## 代码实现

```python
class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []

        # 数字到字母的映射
        phone = {
            '2': 'abc',
            '3': 'def',
            '4': 'ghi',
            '5': 'jkl',
            '6': 'mno',
            '7': 'pqrs',
            '8': 'tuv',
            '9': 'wxyz'
        }

        result = []

        def backtrack(index, path):
            # 终止条件：处理完所有数字
            if index == len(digits):
                result.append(''.join(path))
                return

            # 当前数字对应的字母
            letters = phone[digits[index]]
            for letter in letters:
                path.append(letter)           # 做选择
                backtrack(index + 1, path)   # 递归到下一个数字
                path.pop()                    # 撤销选择

        backtrack(0, [])
        return result
```

---

## 代码关键点解释

| 步骤 | 代码 | 解释 |
|------|------|------|
| 1 | `phone[digits[index]]` | 获取当前数字对应的字母 |
| 2 | `for letter in letters` | 遍历每个字母选择 |
| 3 | `backtrack(index + 1, path)` | 递归到下一个数字 |
| 4 | `''.join(path)` | 路径转字符串 |

---

## 复杂度分析

| 复杂度 | 值 | 说明 |
|--------|---|------|
| **时间** | O(4^n × n) | n 个数字，每个最多 4 个字母 |
| **空间** | O(n) | 递归栈深度 |

## 面试问答

**Q：为什么不用嵌套循环？**

A：因为数字个数不固定，嵌套层数无法确定。回溯法可以处理任意层数的情况。

**Q：空间复杂度为什么是 O(n)？**

A：虽然结果有 4^n 个，但存储结果的空间是必需的。递归栈深度最多为 n（数字个数）。
