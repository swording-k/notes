# LeetCode 27. 移除元素

## 题目链接
https://leetcode.cn/problems/remove-element/

## 题目描述
给定一个整数数组 `nums` 和一个整数 `val`，原地移除所有等于 `val` 的元素。返回移除后数组中不等于 `val` 的元素数量 `k`，且数组前 `k` 个元素应为不等于 `val` 的元素（顺序可以改变）。

## 示例
```
示例 1：
输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2,_ ,_]

示例 2：
输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3,_ ,_ ,_]
```

## 提示
1. 只需将所有等于 val 的元素移到数组末尾，不需要真正"删除"
2. 使用双指针技巧
3. 只需在单次遍历中把保留的元素复制到前面即可

## 约束条件
- 0 ≤ nums.length ≤ 100
- 0 ≤ nums[i] ≤ 50
- 0 ≤ val ≤ 100

## 解题思路：双指针

### 核心思想
- **快慢指针**：fast 遍历原数组，slow 指向保留元素的位置
- 当 `nums[fast] != val` 时，将其复制到 slow 位置

### 图解过程
```
原始数组: [0,1,2,2,3,0,4,2], val = 2

slow=0, fast=0: nums[0]=0 != 2, nums[0]=0, slow=1
slow=1, fast=1: nums[1]=1 != 2, nums[1]=1, slow=2
slow=2, fast=2: nums[2]=2 == 2, 跳过
slow=2, fast=3: nums[3]=2 == 2, 跳过
slow=2, fast=4: nums[4]=3 != 2, nums[2]=3, slow=3
... 以此类推

最终: [0,1,3,0,4,_,_,_], 返回 5
```

### 关键要点
- 不需要真正删除，只需把不等于 val 的元素放到前面
- 返回的是新长度 k，判题系统检查 nums[:k]

## 代码实现

### LeetCode 函数模式
```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != val:
                nums[slow] = nums[fast]
                slow += 1
        return slow
```

### ACM 模式
```python
def remove_element(nums: list[int], val: int) -> int:
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow

if __name__ == "__main__":
    n = int(input())
    nums = list(map(int, input().split()))
    val = int(input())
    k = remove_element(nums, val)
    print(k)
    print(' '.join(map(str, nums[:k])))
```

## 复杂度分析
- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
