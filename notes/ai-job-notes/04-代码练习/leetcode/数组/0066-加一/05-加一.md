# LeetCode 66. 加一

## 题目链接
https://leetcode.cn/problems/plus-one/

## 题目描述
给定一个表示大整数的整数数组 `digits`，其中 `digits[i]` 是整数的第 i 位数字。数组中的数字按从左到右的顺序排列，从最高位到最低位。这个大整数不包含任何前导 0。

将大整数加 1，并返回结果的数字数组。

## 示例
```
示例 1：
输入：digits = [1,2,3]
输出：[1,2,4]
解释：123 + 1 = 124

示例 2：
输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：4321 + 1 = 4322

示例 3：
输入：digits = [9]
输出：[1,0]
解释：9 + 1 = 10
```

## 提示
- 1 ≤ digits.length ≤ 100
- 0 ≤ digits[i] ≤ 9
- digits 不包含任何前导 0

## 解题思路

### 方法：从后往前处理进位

1. 从数组最后一位开始，向前处理
2. 如果当前位小于 9，直接加 1，返回
3. 如果当前位等于 9，设置为本位 0，继续向前处理进位
4. 如果处理完所有位还有进位（如 999 -> 1000），在数组前面插入 1

### 关键要点
- 特殊情况：全是 9 的数组（999 -> 1000）
- 从后往前处理可以利用原地修改
- 最终如果还有进位，需要在前面插入 1

## 代码实现

### LeetCode 函数模式
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits
            digits[i] = 0  # 相当于进位
        # 如果所有位都是 9（如 999 -> 1000）
        return [1] + digits
```

### ACM 模式
```python
def plus_one(digits: list[int]) -> list[int]:
    for i in range(len(digits) - 1, -1, -1):
        if digits[i] < 9:
            digits[i] += 1
            return digits
        digits[i] = 0
    return [1] + digits

if __name__ == "__main__":
    n = int(input())
    digits = list(map(int, input().split()))
    result = plus_one(digits)
    print(' '.join(map(str, result)))
```

## 复杂度分析
- **时间复杂度**：O(n)，最坏情况遍历所有位
- **空间复杂度**：O(1)，不算返回的新数组
