# LeetCode 35. 搜索插入位置

## 题目链接
https://leetcode.cn/problems/search-insert-position/

## 题目描述
给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。**必须使用时间复杂度为 O(log n) 的算法。**

## 示例
```
示例 1：
输入：nums = [1,3,5,6], target = 5
输出：2

示例 2：
输入：nums = [1,3,5,6], target = 2
输出：1

示例 3：
输入：nums = [1,3,5,6], target = 7
输出：4
```

## 提示
- 1 ≤ nums.length ≤ 10⁴
- -10⁴ ≤ nums[i] ≤ 10⁴
- nums 为无重复元素的升序排列数组
- -10⁴ ≤ target ≤ 10⁴

## 解题思路：二分查找

### 核心思想
- 利用有序数组的特性，每次将搜索范围缩小一半
- 当 left > right 时，left 就是目标值应该插入的位置

### 二分查找模板
```python
left, right = 0, len(nums) - 1
while left <= right:
    mid = (left + right) // 2
    if nums[mid] == target:
        return mid
    elif nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
return left  # 插入位置
```

### 关键要点
- **必须返回 left**，不是 right
- 循环条件是 `left <= right`
- 当找不到目标时，left 会停在插入位置

## 代码实现

### LeetCode 函数模式
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        return left
```

### ACM 模式
```python
def search_insert(nums: list[int], target: int) -> int:
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return left

if __name__ == "__main__":
    n = int(input())
    nums = list(map(int, input().split()))
    target = int(input())
    print(search_insert(nums, target))
```

## 复杂度分析
- **时间复杂度**：O(log n)
- **空间复杂度**：O(1)
