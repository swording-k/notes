# LeetCode 19. 删除链表的倒数第N个结点

## 题目链接
https://leetcode.cn/problems/remove-nth-node-from-end-of-list/

## 题目描述
给你一个链表，删除链表的倒数第 n 个节点，返回头节点。

## 示例
```
示例 1：
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

示例 2：
输入：head = [1], n = 1
输出：[]
```

## 解题思路

### 快慢指针 O(n) ✅
```python
class Solution:
    def removeNthFromEnd(self, head: ListNode, n: int) -> ListNode:
        dummy = ListNode(0, head)
        fast = head
        slow = dummy

        # fast 先走 n 步
        for _ in range(n):
            fast = fast.next

        # 一起走，直到 fast 到达末尾
        while fast:
            fast = fast.next
            slow = slow.next

        # 删除节点
        slow.next = slow.next.next

        return dummy.next
```

### 核心思想
- 让 fast 先走 n 步
- 然后 slow 和 fast 一起走
- 当 fast 到达末尾时，slow 的下一个就是要删除的节点

## 复杂度分析
- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
