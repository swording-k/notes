# LeetCode 771. 宝石与石头

## 题目链接
https://leetcode.cn/problems/jewels-and-stones/

## 题目描述
给你两个字符串 `jewels` 和 `stones`：
- `jewels` 代表宝石的类型，每个字符都是一种独特的宝石
- `stones` 代表你拥有的石头，每个字符代表一块石头

统计有多少块石头是宝石。

## 示例
```
示例 1：
输入：jewels = "aA", stones = "aAAbbbb"
输出：3
解释：有 3 块石头(a, A, A)是宝石

示例 2：
输入：jewels = "z", stones = "ZZ"
输出：0
解释：没有石头是宝石
```

## 提示
- 1 ≤ jewels.length ≤ 50
- 1 ≤ stones.length ≤ 1000
- jewels 和 stones 仅由英文字母组成
- jewels 中的所有字符都是唯一的

## 解题思路

### 方法1：set 查找 O(m+n) ✅ 推荐
```python
class Solution:
    def numJewelsInStones(self, jewels: str, stones: str) -> int:
        jewel_set = set(jewels)  # 宝石转 set，O(1) 查找
        count = 0
        for stone in stones:
            if stone in jewel_set:
                count += 1
        return count
```

### 方法2：Counter 统计
```python
from collections import Counter

class Solution:
    def numJewelsInStones(self, jewels: str, stones: str) -> int:
        stone_count = Counter(stones)
        return sum(count for stone, count in stone_count.items() if stone in set(jewels))
```

### 方法3：一行优化
```python
class Solution:
    def numJewelsInStones(self, jewels: str, stones: str) -> int:
        return sum(s in set(jewels) for s in stones)
```

### 核心思想
- 把 jewels 转成 set（因为 jewels 里的字符都是唯一的）
- 遍历 stones，对每个 stone 检查是否在 jewel_set 中
- 计数返回

## 代码实现

### LeetCode 函数模式
```python
class Solution:
    def numJewelsInStones(self, jewels: str, stones: str) -> int:
        jewel_set = set(jewels)
        return sum(1 for stone in stones if stone in jewel_set)
```

### ACM 模式
```python
def num_jewels_in_stones(jewels: str, stones: str) -> int:
    jewel_set = set(jewels)
    return sum(1 for stone in stones if stone in jewel_set)

if __name__ == "__main__":
    jewels = input().strip()
    stones = input().strip()
    print(num_jewels_in_stones(jewels, stones))
```
