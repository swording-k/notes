# LeetCode 217. 存在重复元素

## 题目链接
https://leetcode.cn/problems/contains-duplicate/

## 题目描述
给你一个整数数组 `nums`。如果任一值在数组中出现至少两次，返回 `true`；否则返回 `false`。

## 示例
```
示例 1：
输入：nums = [1,2,3,1]
输出：true
解释：元素 1 出现了两次

示例 2：
输入：nums = [1,2,3,4]
输出：false
解释：所有元素都不重复

示例 3：
输入：nums = [1,1,1,3,3,4,3,2,4,2]
输出：true
解释：有多个重复元素
```

## 提示
- 1 ≤ nums.length ≤ 10⁵
- -10⁹ ≤ nums[i] ≤ 10⁹

## 解题思路

### 方法1：字典计数 O(n)
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        count = {}
        for num in nums:
            if num in count:
                return True
            count[num] = 1
        return False
```

### 方法2：set 去重 O(n) ✅ 推荐
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        seen = set()
        for num in nums:
            if num in seen:
                return True
            seen.add(num)
        return False
```

### 方法3：set 长度对比 O(n)
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        return len(nums) != len(set(nums))
```
一行搞定，但需要额外 O(n) 空间存储 set。

### 核心思想
- 用 set 的 O(1) 查找来判断"是否见过"
- 只要遍历过程中发现重复，直接返回 True

## 代码实现

### LeetCode 函数模式
```python
from typing import List

class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        seen = set()
        for num in nums:
            if num in seen:
                return True
            seen.add(num)
        return False
```

### ACM 模式
```python
def contains_duplicate(nums: list[int]) -> bool:
    seen = set()
    for num in nums:
        if num in seen:
            return True
        seen.add(num)
    return False

if __name__ == "__main__":
    nums = list(map(int, input().split()))
    result = contains_duplicate(nums)
    print("true" if result else "false")
```
