# LeetCode 387. 字符串首个唯一字符

## 题目链接
https://leetcode.cn/problems/first-unique-character-in-a-string/

## 题目描述
给定一个字符串 `s`，找到它的第一个不重复的字符，并返回它的索引。如果不存在，返回 `-1`。

## 示例
```
示例 1：
输入：s = "leetcode"
输出：0
解释：'l' 是第一个不重复的字符

示例 2：
输入：s = "loveleetcode"
输出：2
解释：'v' 是第一个不重复的字符

示例 3：
输入：s = "aabb"
输出：-1
解释：没有不重复的字符
```

## 提示
- 1 ≤ s.length ≤ 10⁵
- s 只包含小写字母

## 解题思路

### 方法1：Counter 统计 + 遍历 O(n) ✅ 推荐
```python
from collections import Counter

class Solution:
    def firstUniqChar(self, s: str) -> int:
        count = Counter(s)  # 统计每个字符出现次数
        for i, ch in enumerate(s):
            if count[ch] == 1:
                return i
        return -1
```

### 方法2：字典手动统计
```python
class Solution:
    def firstUniqChar(self, s: str) -> int:
        count = {}
        for ch in s:
            count[ch] = count.get(ch, 0) + 1
        for i, ch in enumerate(s):
            if count[ch] == 1:
                return i
        return -1
```

### 方法3：队列（保持顺序）
```python
class Solution:
    def firstUniqChar(self, s: str) -> int:
        from collections import deque
        count = {}
        queue = deque()
        for ch in s:
            if ch not in count:
                queue.append(ch)
            count[ch] = count.get(ch, 0) + 1
        while queue:
            ch = queue.popleft()
            if count[ch] == 1:
                return s.index(ch)
        return -1
```

### 核心思想
- **第一遍**：统计每个字符出现的次数
- **第二遍**：按顺序找第一个只出现一次的字符
- 两次遍历，但都是 O(n)，总复杂度 O(n)

### ACM 模式
```python
from collections import Counter
import sys

def first_uniq_char(s: str) -> int:
    count = Counter(s)
    for i, ch in enumerate(s):
        if count[ch] == 1:
            return i
    return -1

if __name__ == "__main__":
    s = sys.stdin.readline().strip()
    print(first_uniq_char(s))
```
