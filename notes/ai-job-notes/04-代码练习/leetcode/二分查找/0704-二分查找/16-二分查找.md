# LeetCode 704. 二分查找

## 题目链接
https://leetcode.cn/problems/binary-search/

## 题目描述
给定一个 n 个元素有序的数组（升序）和一个目标值 target，用 O(log n) 的时间复杂度查找 target。

## 示例
```
示例 1：
输入：nums = [-1,0,3,5,9,12], target = 9
输出：4
解释：9 存在于 nums 中，下标是 4

示例 2：
输入：nums = [-1,0,3,5,9,12], target = 2
输出：-1
解释：2 不存在于数组中
```

## 提示
- 1 ≤ nums.length ≤ 10⁴
- nums 中所有元素不重复
- nums 按升序排列

## 解题思路

### 标准二分查找 O(log n) ✅
```python
class Solution:
    def search(self, nums: list[int], target: int) -> int:
        left, right = 0, len(nums) - 1

        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1

        return -1
```

### 核心思想
- 有序数组，每次取中间比较
- target 大于 mid，搜右半边
- target 小于 mid，搜左半边
- 搜不到就返回 -1

### 易错点
- `while left <= right`：有等号，因为 left==right 时还有一个元素要检查
- `mid = (left + right) // 2`：整数除法，向下取整
- 更新 left/right 时要 ±1，否则会死循环

### ACM 模式
```python
def binary_search(nums: list[int], target: int) -> int:
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

if __name__ == "__main__":
    nums = list(map(int, input().split()))
    target = int(input())
    print(binary_search(nums, target))
```

## 复杂度分析
- **时间复杂度**：O(log n)
- **空间复杂度**：O(1)
