# LeetCode 122. 买卖股票最佳时机 II

> 原题链接：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/

## 题目描述

给定一个数组 `prices`，其中 `prices[i]` 是某支股票第 i 天的价格。

设计算法获取最大利润。可以无限次数地完成交易，但是不能同时参与多笔交易（必须在再次购买前出售掉之前的股票）。

## 示例

```
示例 1：
输入：prices = [7,1,5,3,6,4]
输出：7
解释：
在第 2 天（股票价格=5）买入，第 3 天（股票价格=3）卖出，利润 = 5-1 = 4
在第 3 天买入，第 6 天卖出，利润 = 6-3 = 3
总利润 = 7

示例 2：
输入：prices = [7,6,4,3,1]
输出：0
解释：没有交易完成（所有股价都在下跌）
```

---

## 解题思路

### 方法：贪心算法

**核心思想**：所有上涨区间都参与交易

**贪心策略**：
- 只要明天价格 > 今天价格，今天就买，明天卖
- 等价于：累加所有相邻的正差值

---

## 图解

**示例：prices = [7,1,5,3,6,4]**

```
价格  7   1   5   3   6   4
       ╲  ╱╲  ╱    ╲  ╱
差值    -6  +4  -2  +3  -2

买卖时机：
- 第1天买(1) → 第2天卖(5)：赚 4
- 第3天买(3) → 第4天卖(6)：赚 3
- 总利润 = 7

等价计算：
所有上升区间的和 = (1-7)负忽略 + 5-1 + (3-5)负忽略 + 6-3 + (4-6)负忽略
              = 0 + 4 + 0 + 3 + 0 = 7
```

**为什么这样贪心？**

```
如果第1天买，第4天卖（7→6）：
利润 = 6-7 = -1

但如果第1天买，第2天卖，第3天买，第4天卖：
利润 = (5-1) + (6-3) = 7

分散交易利润 >= 一次交易利润（当有多个上升段时）
```

## 代码实现

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        profit = 0

        for i in range(1, len(prices)):
            if prices[i] > prices[i-1]:
                profit += prices[i] - prices[i-1]

        return profit
```

---

## 代码关键点解释

| 步骤 | 代码 | 解释 |
|------|------|------|
| 1 | `for i in range(1, len(prices))` | 从第二天开始 |
| 2 | `if prices[i] > prices[i-1]` | 如果今天比昨天涨 |
| 3 | `profit += prices[i] - prices[i-1]` | 累加利润 |

## 面试问答

**Q：为什么可以无限次交易？**

A：因为股票卖出后，资金回来了，可以立即买新的。不像有些题目限制最多交易一次或两次。

**Q：贪心为什么正确？**

A：在一个上涨区间 `[a,b]`（a<b）中，任意分割成 `[a,x]` + `[x,b]` 的利润和 = b-a = 直接持有的利润。所以分散交易不会亏损，只要每个上涨都参与就能获得最大利润。

**Q：动态规划怎么解？**
