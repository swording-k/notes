# LeetCode 344. 反转字符串

## 题目链接
https://leetcode.cn/problems/reverse-string/

## 题目描述
编写一个函数，反转输入的字符数组。必须**原地**修改数组，使用 **O(1)** 额外空间。

## 示例
```
示例 1：
输入：s = ["h","e","l","l","o"]
输出：["o","l","l","e","h"]

示例 2：
输入：s = ["H","a","n","n","a","h"]
输出：["h","a","n","n","a","H"]
```

## 约束条件
- 1 ≤ s.length ≤ 10⁵
- s[i] 为可打印 ASCII 字符

## 解题思路

### 方法：双指针交换
- left 指向开头，right 指向结尾
- 交换两个位置的字符
- left++，right--
- 循环直到 left >= right

### 代码实现
```python
class Solution:
    def reverseString(self, s: List[str]) -> None:
        """
        反转字符串（原地修改）
        不能返回任何值，但函数签名要有 List[str] 类型
        """
        left, right = 0, len(s) - 1
        while left < right:
            s[left], s[right] = s[right], s[left]
            left += 1
            right -= 1
```

### Python 最简写法
```python
s.reverse()  # 列表原地反转
```

## 复杂度分析
- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
