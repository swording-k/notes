# LeetCode 20. 有效的括号

## 题目链接
https://leetcode.cn/problems/valid-parentheses/

## 题目描述
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合
2. 左括号必须以正确的顺序闭合
3. 每个右括号都有一个对应的左括号

## 示例
```
示例 1：
输入：s = "()"
输出：true

示例 2：
输入：s = "()[]{}"
输出：true

示例 3：
输入：s = "(]"
输出：false

示例 4：
输入：s = "([)]"
输出：false

示例 5：
输入：s = "{[]}"
输出：true
```

## 提示
- 1 ≤ s.length ≤ 10⁴
- s 仅由括号 '()[]{}' 组成

## 解题思路

### 方法1：栈模拟 O(n) ✅ 推荐
```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        mapping = {')': '(', '}': '{', ']': '['}

        for char in s:
            if char in mapping:
                # 遇到右括号，检查栈顶
                top = stack.pop() if stack else '#'
                if mapping[char] != top:
                    return False
            else:
                # 遇到左括号，入栈
                stack.append(char)

        return len(stack) == 0
```

### 核心思想：栈的特性
- **后进先出（LIFO）**：最后放入的左括号，必须最先匹配
- 遇到左括号就入栈
- 遇到右括号就弹出栈顶检查是否匹配
- 最终栈空说明全部匹配成功

### 匹配过程示例
```
s = "({[]})"
遍历：
  '(' → 入栈 ['(']
  '{' → 入栈 ['(', '{']
  '[' → 入栈 ['(', '{', '[']
  ']' → 匹配 '['，弹出 ['(', '{']
  '}' → 匹配 '{'，弹出 ['(']
  ')' → 匹配 '('，弹出 []
栈空 → true
```

### ACM 模式
```python
def is_valid(s: str) -> bool:
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}

    for char in s:
        if char in mapping:
            top = stack.pop() if stack else '#'
            if mapping[char] != top:
                return False
        else:
            stack.append(char)

    return len(stack) == 0

if __name__ == "__main__":
    s = input().strip()
    print("true" if is_valid(s) else "false")
```

## 复杂度分析
- **时间复杂度**：O(n)
- **空间复杂度**：O(n)

## 易错点
1. 栈可能为空时不能 pop，要先检查
2. 遍历完后栈要为空才说明全部匹配
3. 用 `'#'` 作为空栈时的哨兵值
