# 动态规划

> 求最值 + 重叠子问题 + 最优子结构 = 动态规划

---

## 一、什么是动态规划？

### 生活类比：爬楼梯
```
问题：爬到第10阶有多少种方法？
分析：
- 第10阶 = 第9阶跳1步 + 第8阶跳2步
- 第9阶 = 第8阶跳1步 + 第7阶跳2步
- ...
每个子问题都被重复计算 → 用 DP 存储

DP 思想：保存子问题的答案，避免重复计算
```

### 核心特征
```
1. 最优子结构：大问题的最优解由小问题的最优解构成
2. 重叠子问题：子问题会被多次计算
3. 无后效性：当前决策不影响未来的子问题
```

---

## 二、动态规划五部曲

### 第一步：定义状态（最重要！）
```python
dp[i] = 爬到第 i 阶的方法数
```

### 第二步：找状态转移方程
```python
# 爬楼梯：dp[i] = dp[i-1] + dp[i-2]
dp[i] = min/max(dp[i-1], dp[i-weight] + cost)
```

### 第三步：初始化
```python
dp[0] = 0          # 容量为0，最小代价是0
dp[0] = 1          # 爬到第0阶，有1种方法（起点）
```

### 第四步：确定遍历顺序
```python
# 爬楼梯：从左到右
for i in range(1, n + 1):
    dp[i] = dp[i-1] + dp[i-2]
```

### 第五步：返回结果
```python
return dp[n]
```

## 三、动态规划分类

### 1. 一维 DP（线性）
```python
dp[i] = 问题只涉及 i
# 例如：爬楼梯、打家劫舍、最大子序和
```

### 2. 二维 DP（空间扩展）
```python
dp[i][j] = 涉及两维状态
# 例如：背包问题、路径问题
```

### 3. 状态压缩（空间优化）
```python
# 只需要前一个状态时，可以压缩成一维
dp[j] = min(dp[j], dp[j-weight] + cost)
```

## 五、0-1 背包 vs 完全背包

### 对比表
| 类型 | 物品数量 | 循环方向 | 状态转移 |
|------|---------|---------|---------|
| 0-1 背包 | 只能选一次 | **从大到小** | dp[j] = max(dp[j], dp[j-weight] + cost) |
| 完全背包 | 无限次 | **从小到大** | dp[j] = min(dp[j], dp[j-coin] + 1) |

### 图解区别
```
完全背包（从小到大）：
j=3:  dp[3] = min(inf, dp[0]+5) = 5      ✅ 第一次用
j=6:  dp[6] = min(inf, dp[3]+5) = 10    ✅ dp[3]已更新，可再用

0-1背包（从大到小）：
j=10: dp[10] = ...
j=6:  dp[6] = min(inf, dp[3]+5) = 5     ✅ dp[3]还是旧值
j=3:  dp[3] = min(inf, dp[0]+5) = 5     ✅ 只能选一次
```

## 六、DP 入门模板

### 模板1：一维 DP（爬楼梯）
```python
def climbStairs(n):
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1], dp[2] = 1, 2
    for i in range(3, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]
```

### 模板2：二维 DP（背包）
```python
def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for j in range(capacity + 1):
            if j >= weights[i-1]:
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-weights[i-1]] + values[i-1])
            else:
                dp[i][j] = dp[i-1][j]
    return dp[n][capacity]
```

### 模板3：状态压缩（优化空间）
```python
def coinChange(coins, amount):
    dp = [amount + 1] * (amount + 1)
    dp[0] = 0
    for coin in coins:           # 先循环物品
        for j in range(coin, amount + 1):  # 从小到大
            dp[j] = min(dp[j], dp[j - coin] + 1)
    return dp[amount] if dp[amount] <= amount else -1
```

## 七、常见错误

### 1. 初始化错误
```python
# ❌ 错误：dp[0] 应该初始化为有意义的值
dp = [0] * (n + 1)  # 如果求最小值，应该初始化为 inf
dp[0] = inf         # 容量为0，代价应该很大

# ✅ 正确：根据问题
dp[0] = 0           # 爬楼梯：0阶有1种方法
dp[0] = inf         # 零钱兑换：0元需要0枚硬币
dp[0] = 0           # 背包：0容量价值为0
```

### 2. 循环方向错误
```python
# ❌ 0-1 背包从小到大
for j in range(weight, capacity + 1):  # 会重复选同一个物品！

# ✅ 0-1 背包从大到小
for j in range(capacity, weight - 1, -1):
    dp[j] = max(dp[j], dp[j - weight] + value)
```

### 3. 状态定义错误
```python
# ❌ 不清晰的状态定义
dp[i] = something  # 不知道这个 dp[i] 代表什么

# ✅ 清晰的状态定义
dp[i] = 爬到第 i 阶的最小花费  # 明确知道每个变量的含义
```

## 八、DP vs 其他算法

| 特点 | DP | 贪心 | 回溯 |
|------|-----|------|------|
| 子问题 | 重叠 | 单次 | 枚举 |
| 最优性 | 一定最优 | 局部最优 | 暴力搜索 |
| 时间复杂度 | O(n) - O(n²) | O(n) | 指数级 |
| 适用场景 | 选不选、有序 | 证明局部最优=全局最优 | 排列组合 |
