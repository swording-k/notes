# 二叉树

> 一种每个节点最多有两个子节点的树形数据结构

---

## 一、二叉树基础

### 1.1 结构定义

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val      # 节点值
        self.left = left    # 左子节点
        self.right = right  # 右子节点
```

### 1.2 二叉树类型

| 类型 | 特点 | 应用场景 |
|------|------|---------|
| 满二叉树 | 每层节点数都满 | 理想状态 |
| 完全二叉树 | 最后一层左对齐 | 堆排序 |
| 二叉搜索树 BST | 左 < 根 < 右 | 查找 |
| 平衡二叉树 AVL | 左右子树高度差 ≤ 1 | 高效查找 |

### 1.3 二叉树 vs 数组/链表

| 操作 | 二叉树 | 数组 |
|------|--------|------|
| 查找 | O(log n) ~ O(n) | O(1) |
| 插入/删除 | O(log n) ~ O(n) | O(n) |
| 有序遍历 | 中序遍历 | 二分查找 |

## 二、二叉树遍历

### 2.1 前序遍历（根-左-右）

```python
def preorder(root):
    if not root:
        return
    print(root.val)        # 根
    preorder(root.left)     # 左
    preorder(root.right)   # 右
```

### 2.2 中序遍历（左-根-右）

```python
def inorder(root):
    if not root:
        return
    inorder(root.left)      # 左
    print(root.val)        # 根
    inorder(root.right)     # 右

# BST 中序遍历 → 有序序列
```

### 2.3 后序遍历（左-右-根）

```python
def postorder(root):
    if not root:
        return
    postorder(root.left)    # 左
    postorder(root.right)   # 右
    print(root.val)         # 根
```

## 三、递归遍历的通用模板

### 3.1 分治法模板

```python
def traverse(root):
    if not root:
        return None

    # 1. 先处理当前节点（或跳过）
    # 2. 递归处理左右子树
    # 3. 合并左右子树结果

    left_result = traverse(root.left)
    right_result = traverse(root.right)

    # 合并结果
    return 合并(left_result, right_result)
```

### 3.2 前/中/后序的统一写法

```python
def dfs(root, order):
    if not root:
        return []

    if order == 'pre':      # 根左右
        return [root.val] + dfs(root.left, order) + dfs(root.right, order)
    elif order == 'in':     # 左根右
        return dfs(root.left, order) + [root.val] + dfs(root.right, order)
    elif order == 'post':    # 左右根
        return dfs(root.left, order) + dfs(root.right, order) + [root.val]
```

## 四、层序遍历（BFS）

### 4.1 队列实现

```python
from collections import deque

def levelOrder(root):
    if not root:
        return []

    result = []
    queue = deque([root])

    while queue:
        node = queue.popleft()
        result.append(node.val)

        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)

    return result
```

### 4.2 分层输出

```python
def levelOrder分层(root):
    if not root:
        return []

    result = []
    queue = deque([root])

    while queue:
        level_size = len(queue)  # 当前层节点数
        level = []

        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)

            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)

        result.append(level)

    return result
```

## 五、二叉树的深度/高度

### 5.1 计算二叉树最大深度

```python
def maxDepth(root):
    if not root:
        return 0

    left_depth = maxDepth(root.left)
    right_depth = maxDepth(root.right)

    return max(left_depth, right_depth) + 1
```

### 5.2 计算二叉树最小深度

```python
def minDepth(root):
    if not root:
        return 0

    if not root.left:
        return minDepth(root.right) + 1
    if not root.right:
        return minDepth(root.left) + 1

    return min(minDepth(root.left), minDepth(root.right)) + 1
```

## 六、BST（二叉搜索树）

### 6.1 BST 定义

- 左子树所有节点 **< 根节点**
- 右子树所有节点 **> 根节点**
- 左右子树也是 BST

### 6.2 BST 搜索

```python
def searchBST(root, val):
    if not root or root.val == val:
        return root

    if val < root.val:
        return searchBST(root.left, val)
    else:
        return searchBST(root.right, val)
```

### 6.3 BST 验证

```python
def isValidBST(root, min_val=float('-inf'), max_val=float('inf')):
    if not root:
        return True

    if root.val <= min_val or root.val >= max_val:
        return False

    return (isValidBST(root.left, min_val, root.val) and
            isValidBST(root.right, root.val, max_val))
```

## 七、对称二叉树

### 7.1 判断是否对称

```python
def isSymmetric(root):
    def check(left, right):
        if not left and not right:
            return True
        if not left or not right:
            return False

        return (left.val == right.val and
                check(left.left, right.right) and
                check(left.right, right.left))

    return check(root.left, root.right)
```

**图解**：
```
    1
   / \
  2   2      ← 对称
 / \ / \
3  4 4  3
```

## 八、路径问题

### 8.1 二叉树路径总和

```python
def hasPathSum(root, targetSum):
    if not root:
        return False

    if not root.left and not root.right:
        return root.val == targetSum

    return (hasPathSum(root.left, targetSum - root.val) or
            hasPathSum(root.right, targetSum - root.val))
```

### 8.2 所有路径（前序遍历）

```python
def binaryTreePaths(root):
    result = []

    def dfs(node, path):
        if not node:
            return

        path.append(str(node.val))

        if not node.left and not node.right:
            result.append('->'.join(path))
        else:
            dfs(node.left, path)
            dfs(node.right, path)

        path.pop()

    dfs(root, [])
    return result
```

## 九、构造二叉树

### 9.1 从前序和中序构造

```python
def buildTree(preorder, inorder):
    if not preorder or not inorder:
        return None

    root = TreeNode(preorder[0])
    mid = inorder.index(preorder[0])

    root.left = buildTree(preorder[1:mid+1], inorder[:mid])
    root.right = buildTree(preorder[mid+1:], inorder[mid+1:])

    return root
```

### 9.2 从中序和后序构造

```python
def buildTreeInPost(inorder, postorder):
    if not inorder or not postorder:
        return None

    root = TreeNode(postorder[-1])
    mid = inorder.index(postorder[-1])

    root.left = buildTreeInPost(inorder[:mid], postorder[:mid])
    root.right = buildTreeInPost(inorder[mid+1:], postorder[:-1])

    return root
```

## 十一、递归模板总结

```python
# 二叉树递归三要素：
# 1. 终止条件：if not root: return
# 2. 处理当前层：访问 root.val
# 3. 递归左右：traverse(root.left), traverse(root.right)

def solution(root):
    if not root:
        return 某个值

    # 前序：先处理根
    result = 某种合并(
        处理(root.val),
        solution(root.left),
        solution(root.right)
    )

    return result
```

## 十二、二叉树与递归的区别

| 维度 | 普通递归 | 二叉树递归 |
|------|---------|-----------|
| 终止条件 | 简单条件 | `if not root` |
| 参数传递 | 单参数 | 传递左右子树 |
| 返回值 | 单值 | 树结构或统计值 |
