# 双指针

> 用两个指针配合遍历，解决数组/链表问题

---

## 一、什么是双指针？

### 核心思想
```
普通遍历：i 从 0 到 n
双指针：left 和 right 从两端向中间走
```

### 常见场景
- **两端向中间**：反转、找两数之和
- **同向追赶**：滑动窗口、删除元素
- **快慢指针**：找链表中点、判断环

---

## 二、双指针的三种类型

### 类型1：两端向中间（相向双指针）

#### 应用场景
- 反转字符串
- 验证回文串
- 两数之和 II

#### 代码模板
```python
left, right = 0, len(s) - 1
while left < right:
    # 处理逻辑
    left += 1
    right -= 1
```

#### 示例：验证回文串
```python
def isPalindrome(s):
    # 清理字符串
    cleaned = ''.join(c.lower() for c in s if c.isalnum())

    # 双指针判断
    left, right = 0, len(cleaned) - 1
    while left < right:
        if cleaned[left] != cleaned[right]:
            return False
        left += 1
        right -= 1
    return True
```

---

### 类型2：同向双指针（快慢指针）

#### 应用场景
- 删除排序数组中的重复项
- 移动零
- 滑动窗口

#### 代码模板
```python
slow = 0
for fast in range(len(nums)):
    if should_keep(nums[fast]):
        nums[slow] = nums[fast]
        slow += 1
return slow
```

#### 示例：删除重复项
```python
def removeDuplicates(nums):
    slow = 1  # 第一个元素一定保留
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow - 1]:  # 新元素
            nums[slow] = nums[fast]
            slow += 1
    return slow
```

---

### 类型3：快慢指针（链表）

#### 应用场景
- 找链表中点
- 判断环
- 找环入口

#### 示例：找链表中点
```python
def middleNode(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

## 三、对比：三种双指针

| 类型 | 指针方向 | 速度 | 典型应用 |
|------|---------|------|---------|
| 相向双指针 | 两端向中间 | 追赶 | 回文、反转 |
| 同向双指针 | 同向 | 同速 | 删除、滑动窗口 |
| 快慢指针 | 同向 | 2:1 | 找中点、判断环 |

## 五、双指针 vs 哈希表

| 问题 | 用双指针 | 用哈希表 |
|------|---------|---------|
| 两数之和 | ❌ 有序数组才行 | ✅ 通用 |
| 去重 | ✅ 有序数组 | ✅ 通用 |
| 回文判断 | ✅ 天然适合 | ❌ 不适合 |

## 六、关键技巧

### 1. 何时移动指针？
```python
# 相向：条件满足就移动
while left < right:
    if condition:
        # 处理
        left += 1
    else:
        right -= 1
```

### 2. 何时退出循环？
```python
# 相向：left >= right
# 同向：fast 到达末尾
# 快慢：fast 或 fast.next 为 None
```

### 3. 避免越界
```python
# 快慢指针必须检查
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
```
