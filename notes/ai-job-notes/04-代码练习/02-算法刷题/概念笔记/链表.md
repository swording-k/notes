# 链表

> 一种离散数据结构，通过指针连接节点

---

## 一、链表基础

### 1.1 结构定义

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val    # 节点值
        self.next = next  # 指向下一个节点
```

### 1.2 链表类型

| 类型 | 特点 | 应用场景 |
|------|------|---------|
| 单向链表 | 只含next指针 | 最基础 |
| 双向链表 | 含prev和next | LRU缓存 |
| 循环链表 | 尾节点指向头 | 约瑟夫问题 |

### 1.3 数组 vs 链表

| 操作 | 数组 | 链表 |
|------|------|------|
| 访问 | O(1) | O(n) |
| 插入/删除(头部) | O(n) | O(1) |
| 插入/删除(尾部) | O(1) | O(n) |
| 内存 | 连续 | 离散 |

## 二、链表遍历

### 2.1 基础遍历

```python
def traverse(head):
    cur = head
    while cur:
        print(cur.val)
        cur = cur.next
```

### 2.2 虚拟头节点（哨兵节点/Sentinel）

#### 为什么需要虚拟头节点？

**核心问题**：删除头节点时，没有"前一个节点"可以指向

```
删除头节点的情况：
[1] -> [2] -> [3] -> None，删除倒数第3个（删除头节点）

如果没有 dummy：
result = head = [1]
result.next = result.next.next  → [1].next = [2]
结果：[1] 还在，变成了 [1] -> [2] -> [3]，根本没删掉！

如果有 dummy：
dummy -> [1] -> [2] -> [3] -> None
         ↑
       result（指向 dummy）

result.next = result.next.next  → dummy.next = [2]
结果：删掉了 [1]，变成 [2] -> [3] ✓
```

#### 虚拟头节点的作用

1. **保证始终有"前一个节点"**：无论删除哪个节点（包括头节点），都有前一个节点可操作
2. **统一代码逻辑**：不需要写 `if result == head` 这样的特殊判断

#### 什么时候必须用 dummy？

| 场景 | 需要 dummy | 原因 |
|------|-----------|------|
| 删除头节点 | ✅ 必须 | 没有前一个节点 |
| 删除中间节点 | ✅ 建议 | 代码更统一 |
| 删除尾节点 | ✅ 建议 | 统一处理 |
| 只遍历/查找 | ❌ 不需要 | 不涉及删除 |

#### 代码模板

```python
def delete_node(head, target):
    # 创建虚拟头节点，指向真正的头节点
    dummy = ListNode(0, head)

    # 用 prev 遍历，prev 始终是"待删节点的前一个"
    prev = dummy
    while prev.next:
        if prev.next.val == target:
            prev.next = prev.next.next  # 删除
            break
        prev = prev.next

    return dummy.next  # 返回真正的头节点
```

#### 图解 dummy 的删除过程

```
初始：[1] -> [2] -> [3] -> None
      dummy -> [1]

删 [2]：
dummy -> [1] -> [2] -> [3]
         ↑prev
prev.next = prev.next.next
dummy -> [1] ------> [3]
```

## 三、ACM模式：链表输入输出

### 3.1 为什么链表不能直接输入？

链表是**离散结构**，不像数组可以用下标访问。输入时必须手动构建。

### 3.2 输入构建链表

```python
# 输入：1 2 3 4 5（用空格分隔的整数）
# 构建：1 -> 2 -> 3 -> 4 -> 5 -> None

vals = list(map(int, input().split()))  # 读取数组 [1,2,3,4,5]

# 构建链表
dummy = ListNode(0)  # 虚拟头节点
cur = dummy
for v in vals:
    cur.next = ListNode(v)  # 创建新节点，连接到链表
    cur = cur.next          # 移动到尾部

head = dummy.next  # 真正的头节点
```

### 3.3 输出链表（转数组）

```python
# 将链表转为数组输出
def linked_list_to_array(head):
    result = []
    cur = head
    while cur:
        result.append(cur.val)
        cur = cur.next
    return result

# 输出：print(*result) 打印 1 2 3 4 5
```

### 3.4 完整ACM模板

```python
from typing import Optional

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        fast = head
        for _ in range(n):
            fast = fast.next
        slow = dummy
        while fast:
            slow = slow.next
            fast = fast.next
        slow.next = slow.next.next
        return dummy.next

# ============ ACM 入口 ============
if __name__ == "__main__":
    # 1. 读取输入（假设第一行是链表值，第二行是n）
    vals = list(map(int, input().split()))  # "1 2 3 4 5"
    n = int(input())                        # "2"

    # 2. 构建链表
    dummy = ListNode(0)
    cur = dummy
    for v in vals:
        cur.next = ListNode(v)
        cur = cur.next
    head = dummy.next

    # 3. 调用算法
    sol = Solution()
    result = sol.removeNthFromEnd(head, n)

    # 4. 输出（链表转数组）
    output = []
    while result:
        output.append(result.val)
        result = result.next
    print(*output)  # "1 2 3 5"
```

### 3.5 边界情况处理

| 情况 | 处理方式 |
|------|---------|
| 空输入 | `if not vals: return` |
| 只有一个节点 | dummy 仍能正确处理 |
| n == 链表长度 | 删除头节点，dummy 仍能正确处理 |

## 四、双指针在链表中的应用

### 4.1 快慢指针基础

```python
def find_middle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

### 4.2 快慢指针的三个经典应用

| 应用 | 算法 | 时间 | 空间 |
|------|------|------|------|
| 找链表中点 | 快走2步，慢走1步 | O(n) | O(1) |
| 判断是否有环 | 快走2步，慢走1步，相遇则有环 | O(n) | O(1) |
| 找环入口 | Floyd算法 | O(n) | O(1) |

### 4.3 公式证明

```
设环入口距离起点为 a，慢指针入环后走了 b，与快指针相遇时快指针多走了 n 圈
2(a + b) = a + b + n(环长)
a + b = n(环长)
a = n(环长) - b
结论：从相遇点走 a 步 = 从起点走 a 步
```

## 五、链表翻转

### 5.1 原地翻转（迭代）

```python
def reverseList(head):
    prev = None
    cur = head
    while cur:
        next_temp = cur.next  # 保存下一个
        cur.next = prev       # 反转指针
        prev = cur            # 移动prev
        cur = next_temp       # 移动cur
    return prev
```

### 5.2 翻转过程图解

```
初始: 1 -> 2 -> 3 -> None
      prev=None, cur=1

Step1: 1 -> None, prev=1, cur=2
Step2: 2 -> 1 -> None, prev=2, cur=3
Step3: 3 -> 2 -> 1 -> None, prev=3, cur=None
返回 prev = 3
```

## 六、链表解题技巧

### 技巧1：虚拟头节点

```python
dummy = ListNode(0, head)
# 操作dummy.next开始的链表
return dummy.next
```

**适用场景**：删除节点、添加节点（统一所有节点的操作用）

### 技巧2：快慢指针

```python
slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
```

**适用场景**：找中点、判断环、找环入口

### 技巧3：穿针引线（反转链表）

```python
prev, cur = None, head
while cur:
    nxt = cur.next
    cur.next = prev
    prev = cur
    cur = nxt
```

### 技巧4：先走N步

```python
fast = head
for _ in range(n):
    fast = fast.next
while fast:
    fast = fast.next
    slow = slow.next
```

**适用场景**：删除倒数第N个节点

## 七、链表四道必刷题

| # | 题目 | 核心技巧 | 难度 |
|---|------|---------|------|

| [[0206-反转链表]] | 反转整个链表 | 三指针翻转 | Easy |
| [[0019-删除链表的倒数第N个结点]] | 删除倒数第N个 | 快慢指针 | Medium |
| [[0142-环形链表II]] | 找环入口 | Floyd算法 | Medium |

### 7.1 移除链表元素

```python
class Solution:
    def removeElements(self, head, val):
        dummy = ListNode(0, head)  # 虚拟头节点
        cur = dummy
        while cur.next:
            if cur.next.val == val:
                cur.next = cur.next.next
            else:
                cur = cur.next
        return dummy.next
```

**关键点**：虚拟头节点统一处理，不用单独判断head

### 7.2 反转链表

```python
class Solution:
    def reverseList(self, head):
        prev = None
        cur = head
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt
        return prev
```

**关键点**：三指针 prev、cur、nxt，逐个反转

### 7.3 删除倒数第N个节点

```python
class Solution:
    def removeNthFromEnd(self, head, n):
        dummy = ListNode(0, head)
        fast = slow = dummy
        for _ in range(n + 1):  # 先走n+1步
            fast = fast.next
        while fast:
            fast = fast.next
            slow = slow.next
        slow.next = slow.next.next  # 删除
        return dummy.next
```

**关键点**：快慢间隔n个节点，快到末尾时慢刚好在待删节点前

### 7.4 环形链表 II

```python
class Solution:
    def detectCycle(self, head):
        fast = slow = head
        while fast and fast.next:
            fast = fast.next.next
            slow = slow.next
            if fast == slow:  # 相遇
                fast = head
                while fast != slow:
                    fast = fast.next
                    slow = slow.next
                return slow
        return None
```

**关键点**：Floyd算法，证明见上方公式

## 八、常见错误

### 错误1：修改指针后丢失后继节点

```python
# 错误：先改了next再访问
cur.next = prev
cur = cur.next  # 已经不在原链表上了！

# 正确：先保存
next_temp = cur.next
cur.next = prev
cur = next_temp
```

### 错误2：循环条件不完整

```python
# 错误：没有检查fast.next
while fast:
    fast = fast.next.next  # 可能报错

# 正确：检查两个
while fast and fast.next:
    fast = fast.next.next
```

### 错误3：未处理空链表

```python
if not head:
    return None
```

## 九、链表题解题框架

```
1. 判断是否为边界条件（空链表、单节点）
2. 是否需要虚拟头节点？（删除/添加头部节点）
3. 用什么遍历方式？
   - 普通遍历：计数、打印
   - 快慢指针：找中点、判断环
   - 先走N步：删除倒数第N个
4. 操作完成后返回正确的head
```
