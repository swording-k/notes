## 📝 常见问题

### Q: 什么时候用哈希表？
```
需要 O(1) 查找/去重/统计频率时
例如：两数之和、找重复字符、统计出现次数
```

### Q: 什么时候用双指针？
```
数组两端向中间逼近、两数之和、链表找环
```

### Q: 什么时候用动态规划？
```
求最值 + 有重叠子问题 + 无后效性（选不选的状态）
例如：最短路径、最大价值、计数问题
```
