### 1.AI聊天 + 计划修改，这就是使用了agent的执行逻辑，

用户："把深蹲换成腿举"

&#x20;       ↓

AI 解析意图 →

&#x20;返回 JSON 指 令：
{
&#x20; "type": "update\_plan",
&#x20; "actions": \[{
&#x20;   "day": 1,
&#x20;   "old\_exercise": "深蹲",
&#x20;   "new\_exercise": "腿屈伸",
&#x20;   "reason": "膝盖友好的替代动作"
&#x20; }]
}

&#x20;       ↓

App 解析 JSON → 执行修改 → 保存到本地数据库

- JSON 指令 = 工具调用
- 解析执行 = Action 执行
