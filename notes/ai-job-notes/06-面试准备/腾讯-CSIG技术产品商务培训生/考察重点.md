## 二、群面考察什么？

### 群面评分维度

| 维度 | 权重 | 具体表现 |
|------|------|----------|
| **逻辑思维** | 30% | 发言有没有条理，方案有没有框架 |
| **表达能力** | 25% | 说话清不清楚，能不能把观点讲明白 |
| **团队协作** | 20% | 能不能配合别人，推动讨论进行 |
| **创新思维** | 15% | 有没有新观点，能不能提出好点子 |
| **个人特质** | 10% | 性格是否适合做销售，有没有感染力 |

### 群面中的加分项

```
✅ 第一个发言，且说得很清楚
✅ 提出清晰的讨论框架
✅ 主动帮助落后组员（"XX同学，你也来说说"）
✅ 有效控制时间（"现在还剩10分钟"）
✅ 总结归纳全组观点
✅ 提出有新意的观点

❌ 全程不说话
❌ 打断别人
❌ 否定别人方案但不提替代建议
❌ 超时
❌ 和别人争吵
```

## 三、单面考察什么？

### 单面问题类型

| 类型 | 出现概率 | 举例 |
|------|----------|------|
| **自我介绍** | 100% | 必问 |
| **Why Sales** | 90% | 为什么做销售 |
| **Why Tencent** | 80% | 为什么选腾讯 |
| **经历挖掘** | 70% | 项目介绍、困难经历 |
| **商业理解** | 60% | 云计算理解、ToB理解 |
| **个人特质** | 50% | 优势劣势、职业规划 |
| **情景应对** | 40% | 客户拒绝怎么办 |

### 每个问题的考察点

```
Q1：自我介绍
   → 考察：表达能力、自我认知、和岗位的匹配度

Q2：你为什么想做销售？
   → 考察：求职动机、对自己是否有规划

Q3：销售最重要的素质是什么？
   → 考察：是否理解销售工作、有没有正确认知

Q4：你对云计算怎么理解？
   → 考察：学习能力、行业认知

Q5：介绍一个你做过的项目
   → 考察：经历真实性、能力水平、思考方式

Q6：你有什么优势/劣势？
   → 考察：自我认知、成长潜力

Q7：你的职业规划是什么？
   → 考察：是否有规划、是否稳定
```

## 四、你必须掌握的知识

### 云计算基础（必背）

```
1️⃣ 什么是云计算：
   - 用"租"代替"买"的模式
   - 按需付费，弹性扩容

2️⃣ 三种服务模式：
   - IaaS：租服务器（给你毛坯房）
   - PaaS：租开发平台（给你简装房）
   - SaaS：租软件（给你精装房）

3️⃣ 企业为什么用云：
   - 省钱：不用买服务器
   - 省心：不用自己维护
   - 弹性：业务高峰快速扩容

4️⃣ 腾讯云核心产品：
   - 云服务器 CVM
   - 云数据库 MySQL/Redis
   - 企业微信、腾讯会议
   - 安全产品：防火墙、DDoS防护
   - 音视频：直播、点播、实时音视频
```

### 腾讯云优势（必背）

```
1️⃣ 社交生态：
   企业微信、微信小程序可以和腾讯云无缝打通

2️⃣ 音视频领先：
   腾讯会议、直播技术是业界最强的

3️⃣ 安全可靠：
   腾讯安全有七大实验室

4️⃣ 全球化布局：
   全球有多个数据中心

5️⃣ 价格灵活：
   按量付费、包年包月可选
```

### ToB销售基础概念

```
1️⃣ ToB和ToC的区别：
   - ToB：企业客户，决策周期长，金额大，需要方案+服务
   - ToC：个人客户，决策周期短，金额小，重产品+体验

2️⃣ 销售的核心能力：
   - 洞察需求：看透客户真正需要什么
   - 建立信任：让客户相信你这个人
   - 价值呈现：把产品价值讲清楚
   - 抗压能力：被拒绝后继续努力

3️⃣ 客户常见的异议：
   - "太贵了" → 算ROI，展示长期价值
   - "考虑一下" → 了解顾虑，继续跟进
   - "选别家了" → 保持关系，等下次机会
```

## 五、情景题万能框架

### 遇到任何情景题，按这个回答：

```
📝 万能框架（5步走）：

1️⃣ 理解问题（10秒）：
   "这个问题核心是..."

2️⃣ 分析现状（20秒）：
   "当前痛点是...，资源有..."

3️⃣ 提出方案（60秒）：
   "我的建议是：
   - 短期：先做...
   - 中期：再...
   - 长期：系统性地..."

4️⃣ 执行路径（15秒）：
   "具体执行：
   - 谁来做？
   - 什么时间节点？
   - 如何衡量成功？"

5️⃣ 风险预判（15秒）：
   "可能遇到的问题：
   - 如果XX情况发生，我会...
   - 备选方案是..."
```

### 具体情景的答案模板

#### 情景：客户说"太贵了"

```
✅ 标准答案：
"客户说贵可能有几个原因：
1️⃣ 不了解价值：重点讲我们能帮他解决什么问题
2️⃣ 预算受限：提供分期方案或精简版
3️⃣ 在比价：强调我们的差异化价值（服务、稳定性）

具体我会：
1. 先了解客户的预算和核心需求
2. 算ROI，展示长期价值
3. 提供几个方案让客户选择
4. 如果是大客户，申请优惠"
```

#### 情景：客户说"考虑一下"

```
✅ 标准答案：
"客户说考虑，通常意味着：
1️⃣ 还没完全信任你
2️⃣ 有顾虑没说出口
3️⃣ 需要更多信息

具体我会：
1. 问清楚顾虑是什么："您是在考虑什么呢？"
2. 针对性地解决
3. 提供案例证明："XX客户一开始也这样想，后来..."
4. 约定下次跟进时间
5. 保持联系，不放弃"
```

#### 情景：被面试官质疑方案

```
✅ 标准答案：
"您的质疑很有道理，我承认方案有不足...

但我想说明的是：
1. 我的思考角度是...
2. 我考虑了这个因素...
3. 如果改进，我会...

您觉得这个方向对吗？"
```
