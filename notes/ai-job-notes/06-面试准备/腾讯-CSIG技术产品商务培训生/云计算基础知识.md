## 一、什么是云计算？

### 一句话理解

```
传统方式：自己买服务器、自己维护、自己升级
云方式：租云服务商（腾讯云/阿里云）的服务器，按需付费

类比：
- 传统 = 自己买房
- 云 = 租房子（按月付费，随时换）
```

## 二、三种服务模式（必背）

| 模式 | 比喻 | 举例 |
|------|------|------|
| **IaaS** | 毛坯房（给你基础设施） | 租服务器、存储、网络 |
| **PaaS** | 简装房（给你开发平台） | 数据库、开发工具、API |
| **SaaS** | 精装房（直接用软件） | 腾讯会议、企业微信、钉钉 |

```
记忆口诀：
I = Infrastructure = 基础设施
P = Platform = 平台
S = Software = 软件
越往右，服务越多，你做的工作越少
```

## 四、ToB 销售 vs ToC 销售

| 维度 | ToB（对企业） | ToC（对个人） |
|------|---------------|---------------|
| 客户 | 企业/商家 | 个人消费者 |
| 决策人 | 通常多人（采购、IT、业务） | 个人 |
| 决策周期 | 长（几周到几个月） | 短（几分钟到几天） |
| 金额 | 大（几万到几百万） | 小（几十到几千） |
| 关系 | 重销售+服务 | 重产品+体验 |
| 关键 | 信任+方案+服务 | 流量+转化 |

```
面试回答示例：
"我认为ToB销售更需要建立长期信任关系，
因为客户买的是解决方案，不是一个产品。
需要深入了解客户痛点，定制方案，持续服务。"
```

## 五、腾讯云 vs 阿里云

### 腾讯云优势

```
1. 社交优势：微信/QQ用户基础，企业微信生态
2. 游戏领域：云游戏、直播解决方案领先
3. 音视频：腾讯会议、实时音视频技术强
4. 安全：安全防护能力业界领先
5. 生态：微信小程序、公众号等可以无缝对接
```

### 阿里云优势（也要知道）

```
1. 电商生态：淘宝、天猫商家需要云服务
2. 计算能力：飞天操作系统，底层技术强
3. 企业市场：政企客户占比较高
4. 生态布局：钉钉、饿了么等企业协同工具
```

### 面试回答

```
"腾讯云的优势在于社交生态和音视频能力，
比如企业微信可以和腾讯云产品无缝打通，
这对很多企业很有吸引力。"
```

## 六、销售核心概念

### 1. 商机挖掘

```
定义：发现客户有购买云产品的需求

方法：
- 和客户聊业务痛点（"你们现在服务器运维压力大吗？"）
- 关注行业趋势（政策变化、新业务上线）
- 竞品动态（客户在用别家云，可以问为什么）
```

### 2. 客户分层

```
A类客户（大客户）：高价值，重点维护
B类客户（中客户）：潜力大，需要培育
C类客户（小客户）：标准化服务
```

### 3. KPI 指标

```
销售常用指标：
- 签约金额（合同多少钱）
- 续费率（老客户是否继续用）
- 商机数量（有多少潜在客户）
- 转化率（商机→签约的比例）
```

## 七、常见销售情景题

### 题目1：客户嫌贵怎么办？

```
回答框架：
1. 先问清楚：客户的预算是多少？对比的目标是什么？
2. 价值拆解：我们贵在哪里？能不能帮客户省钱/赚钱？
3. 分期方案：可以先买核心功能，后面再扩展
4. 案例说服：举一个客户用了我们产品ROI提升的案例
```

### 题目2：客户说"在考虑阿里云"

```
回答框架：
1. 认可竞品：不贬低阿里云，那是成熟的对手
2. 差异化：腾讯云的优势是什么？腾讯会议、企业微信等
3. 需求匹配：客户的核心需求是什么？我们更匹配什么？
4. 信任建立：先试用、让客户看到效果
```

### 题目3：客户需求和我们的产品不匹配

```
回答框架：
1. 确认需求：是真的不匹配还是你理解错了？
2. 寻找替代：有没有类似的产品能满足？
3. 反馈产品：记录下来，反馈给产品团队
4. 长期跟进：即使这次不成，也保持关系等机会
```

4. 长期发展：销售做好了可以转型产品、市场、管理
