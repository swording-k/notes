## 二、项目经理岗位知识

### 2.1 项目管理基础流程

```
项目生命周期：
启动 → 规划 → 执行 → 监控 → 收尾
  ↓       ↓       ↓       ↓       ↓
立项    分解任务  落地    追踪    复盘
```

### 2.2 项目经理核心能力（按重要程度）

| 能力 | 解释 |
|------|------|
| **跨部门沟通** | 对接研发、设计、运营、供应链 |
| **时间管理** | 优先级判断、deadline控制 |
| **风险管理** | 提前预判问题，制定预案 |
| **数据分析** | 用数据做决策，而非拍脑袋 |
| **执行力** | 把事情做成，不找借口 |

### 2.3 项目经理 vs 产品经理

| 项目经理 | 产品经理 |
|----------|----------|
| 关注"怎么做成" | 关注"做什么" |
| 交付结果 | 定义方向 |
| 协调资源 | 定义需求 |
| 执行导向 | 战略导向 |

**注意**：安克的"项目经理"更偏向**项目交付**，而非产品规划。

### 2.4 常见面试问题（必须准备）

**Q1：你怎么理解项目经理这个岗位？**

```
答：项目经理是"让事情发生的人"。
- 把模糊的需求变成可落地的计划
- 协调各方资源，解决冲突
- 按时按质交付结果
- 风险管理，提前预警
```

**Q2：跨部门协作遇到冲突怎么办？**

```
答：三步走：
1. 理解对方立场（为什么他不同意？）
2. 找到共同目标（我们都想把项目做成）
3. 提出折中方案（找到双方都能接受的解）
```

**Q3：项目延期了怎么办？**

```
答：诚实 + 行动
1. 第一时间同步风险，不掩盖
2. 分析延期原因
3. 制定补救计划
4. 后续避免类似问题
```

## 三、跨境电商知识（安克特色）

### 3.1 亚马逊运营核心概念

| 术语 | 解释 |
|------|------|
| **FBA** | Fulfillment by Amazon，亚马逊仓储物流 |
| **BSR** | Best Seller Rank，销量排名 |
| **CPC** | 点击付费广告 |
| **Review** | 用户评价（对转化率影响大） |
| **Buy Box** | 黄金购物车，获得更多曝光 |

### 3.2 出海品牌打法

```
安克的品牌之路：
1. 阶段一：白牌代工（2011年前）
2. 阶段二：品质提升 + 亚马逊起步
3. 阶段三：打造自有品牌 Anker
4. 阶段四：多品牌扩张（Soundcore、Eufy）
5. 阶段五：线下渠道拓展
```

### 3.3 跨境电商 vs 国内电商

| 维度 | 跨境电商 | 国内电商 |
|------|----------|----------|
| **物流** | 长周期、海关 | 国内快递 |
| **用户** | 海外消费者 | 中国消费者 |
| **平台** | 亚马逊、eBay | 淘宝、京东 |
| **语言** | 多语言 | 中文 |
| **本地化** | 必须做 | 相对简单 |
