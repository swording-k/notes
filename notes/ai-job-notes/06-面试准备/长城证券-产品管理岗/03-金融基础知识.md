 # 金融基础知识（券商方向）

> 30分钟快速建立金融认知

## 一、证券公司是什么？

```
证券公司（券商）= 经营证券业务的金融机构
主要业务:
- 帮人买卖股票（经纪业务）
- 帮人发行股票（投行业务）
- 自己投资（自营业务）
- 管理别人的钱（资管业务）
```

## 二、券商的主要业务线

### 1. 财富管理（最贴近C端）
```
业务: 帮客户开户、交易、提供投资建议
收入: 佣金、咨询费
C端产品: 券商APP、交易软件
代表App: 同花顺、东方财富、各大券商APP
```

### 2. 投资银行（最赚钱）
```
业务: 帮公司上市、发行债券、并购重组
收入: 承销费、顾问费
B端服务: 公司客户
代表业务: IPO、再融资、并购
```

### 3. 资产管理
```
业务: 发理财产品帮客户管理钱
收入: 管理费、业绩分成
产品: 集合资管、定向资管、专项资管
```

### 4. 自营投资
```
业务: 用自己的钱投资
收入: 投资收益
风险: 亏了算公司的
```

### 5. 研究咨询
```
业务: 行业研究、股票研究
收入: 佣金分仓、报告销售
代表: 各大券商研究所
```

## 三、证券业务核心概念

### 1. A股市场
```
A股: 人民币普通股票，在中国境内上市
主要板块:
- 主板（000xxx）：成熟大公司
- 创业板（300xxx）：成长型公司
- 科创板（688xxx）：科技创新公司
- 北交所（8xxxxx）：中小创新企业
```

### 2. 交易流程
```
开户 → 入金 → 选股 → 下单 → 撮合 → 清算 → 交割
每个环节都可能成为产品优化的点
```

### 3. 投资品种
```
- 股票（A股、港股、美股）
- 债券（国债、企业债、可转债）
- 基金（公募、私募）
- 衍生品（期货、期权）
- 另类投资（私募股权、不动产）
```

## 四、券商IT系统的核心模块

### 1. 核心交易系统
```
作用: 撮合买卖订单
特点: 高并发、低延迟、稳定
技术栈: C++、Java、FPGA
代表厂商: 金证、恒生、金仕达
```

### 2. 客户管理系统（CRM）
```
作用: 管理客户信息、资产、交易
特点: 数据量大、查询频繁
产品价值: 客户运营、精准营销
```

### 3. 风控系统
```
作用: 监控异常交易、风险预警
特点: 实时性、准确性
监管要求: 合规先行
```

### 4. 网上交易系统
```
作用: 客户通过手机/电脑交易
特点: 用户体验要求高
创新点: AI选股、智能投顾
```

### 5. 资管系统
```
作用: 理财产品管理
特点: 净值计算、份额管理
```

## 五、金融科技（FinTech）的核心趋势

### 1. 数字化转型
```
传统模式: 线下网点、人工服务
数字化: 手机开户、在线交易、智能客服
代表: 招商银行、平安银行
```

### 2. 智能化升级
```
传统模式: 人工投顾、经验决策
智能化: 智能投顾、量化交易
代表: 同花顺i问财、东方财富Choice
```

### 3. 平台化建设
```
传统模式: 业务系统独立
平台化: 中台架构、统一服务
代表: 蚂蚁集团、京东数科
```

### 4. 移动化
```
传统模式: PC端为主
移动化: APP为主、小程序为辅
代表: 各大券商APP
```

## 六、券商行业竞争格局

### 第一梯队（头部券商）
```
中信证券、华泰证券、海通证券、国泰君安
特点: 综合实力强、业务全面
```

### 第二梯队（大型券商）
```
招商证券、广发证券、国信证券、长城证券（部分）
特点: 业务有特色
```

### 第三梯队（中小券商）
```
特点: 业务相对单一、区域性强
```

### 互联网券商（新兴）
```
东方财富、华泰证券（涨乐财富通）
特点: 互联网思维、用户体验好
```

## 八、面试可能问的金融问题

### Q1: 你对券商业务了解多少？
> "券商主要有五大业务线：财富管理（帮客户交易）、投行（帮公司上市）、资管（发产品）、自营（自己投资）、研究咨询。长城证券作为中型综合券商，业务覆盖比较全面。"

### Q2: 你知道长城证券的优势业务吗？
> "长城证券依托华能集团，在能源金融方面有特色，是综合类证券公司，业务覆盖财富管理、投行、资管等。作为国资背景的券商，长城证券在能源行业投融资服务方面有独特优势。"

### Q3: 券商APP的核心功能是什么？
> "开户、交易、行情、资讯、理财这五大核心模块。同花顺、东方财富等是行业标杆，长城证券自己的APP也在持续升级。"

### Q4: 金融科技对券商有什么影响？
> "金融科技正在重塑券商业务模式：开户从线下转到线上、投顾从人工转向智能、风控从经验转向数据、AI正在改变投研、运营等各环节。"
