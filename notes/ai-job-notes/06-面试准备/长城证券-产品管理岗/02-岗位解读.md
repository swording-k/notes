# 长城证券 岗位深度解读

> 逐条拆解JD，挖掘面试考点

## 岗位职责逐条分析

### 职责1：需求调研与产品设计

```
原文:
"深入理解业务场景与用户需求，完成需求分析、竞品调研和产品方案设计，输出高质量的 PRD 文档和原型。"

拆解:
- 业务场景理解 → 证券业务知识
- 用户需求分析 → 用户研究能力
- 竞品调研 → 同花顺、东方财富、Wind等
- PRD文档 → 产品专业能力
- 原型设计 → Axure/Figma

面试可能问:
- 你怎么理解"业务场景"？
- 你做过竞品分析吗？方法是什么？
- 怎么写一份PRD？
```

### 职责2：项目推进与跨团队协同

```
原文:
"作为项目核心驱动者，协调前端、后端、测试等多方资源，制定项目计划并跟踪落地，确保产品按时交付并达成预期目标。"

拆解:
- 项目核心驱动者 → 推动力
- 跨团队协同 → 沟通协调能力
- 项目计划制定 → 项目管理能力
- 按时交付 → 执行力

面试可能问:
- 项目delay了怎么办？
- 怎么协调技术和业务的冲突？
- 跨团队沟通有什么技巧？
```

### 职责3：运营产品与客户赋能

```
原文:
"指导客户正确、高效地使用产品，梳理并输出产品使用指南和最佳实践；通过分析客户使用情况，发现产品短板并提出改进建议，持续提升产品使用效率与准确度。"

拆解:
- 客户赋能 → 培训能力
- 使用指南 → 文档能力
- 客户使用分析 → 数据分析
- 产品短板 → 优化迭代

面试可能问:
- 怎么衡量"客户赋能"效果？
- 你怎么从客户反馈中发现产品问题？
```

### 职责4：数据驱动与AI工具探索 ⭐⭐⭐

```
原文:
"关注产品上线后的数据表现，通过量化分析持续优化产品体验；主动探索 AI 工具在产品设计、用户研究等环节的应用，推动团队智能化工作流建设。"

拆解:
- 数据分析能力 → 必须展示
- 量化分析方法 → A/B测试、漏斗分析
- AI工具探索 → 你的强项！重点准备
- 智能化工作流 → 团队提效

这是JD中你最有优势的地方，面试时必须重点表现！
```

```
你的情况: 中南大学计算机科班 ✅ 完美命中
```

### 资格2：专业能力

```
要求:
- 较强的逻辑分析能力 ✅ 你有
```

```
要求:
```

## 面试官可能问的问题

### 业务类
1. 你对证券业务了解多少？
2. 同花顺、东方财富、Wind有什么区别？
3. 你觉得券商APP最需要改进的是什么？

### 产品类
1. 怎么写一份PRD？请举例
2. 你做过竞品分析吗？方法是什么？
3. 如果让你设计一个券商的"智能投顾"功能，你会怎么做？

### AI类 ⭐⭐⭐
1. AI可以在券商哪些场景落地？
2. 你用过哪些AI工具？效果如何？
3. 怎么用AI提升产品设计效率？
4. 怎么用AI做用户研究？

### 软实力类
1. 项目delay了怎么办？
2. 技术和业务有冲突怎么协调？
3. 你的最大优势是什么？
