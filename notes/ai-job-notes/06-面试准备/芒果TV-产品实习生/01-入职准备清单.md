|------|------|

## 二、心态准备

```
"无薪资不是亏，是投资"的心态
- 接受现状，但不代表躺平
- 每周都问自己: 今天学到了什么新东西？
- 定期和leader沟通，了解自己在团队的成长路径
```

## 四、第一周要做的事

### Day1-2：观察期
```
✅ 不要急着表现，先观察团队的工作方式
✅ 主动问同事问题（产品组的、设计组的、研发组的）
✅ 记下每个人的名字和职责
✅ 了解团队的OKR和当前在做什么项目
```

### Day3-5：建立关系
```
✅ 主动约1对1咖啡时间
✅ 问leader: "我前两周最需要做什么？"
✅ 问mentor: "团队最缺什么能力的人？"
✅ 加入团队群，了解日常沟通方式
```

## 五、第一周必问的10个问题

### 对Leader
1. 这个岗位3个月后最希望我达成什么？
2. 我的mentor是谁？遇到问题可以找谁？
3. 团队现在最头疼的问题是什么？
4. 哪些业务线我可以参与？

### 对Mentor
5. 我每天的工作流程是怎样的？
6. 产品文档怎么写？有什么模板？
7. 怎么参与需求评审？
8. 怎么提需求？流程是什么？

### 对自己
9. 我应该重点关注哪些业务数据？
10. 怎么证明自己的价值？

## 六、30秒自我介绍模板

> "大家好，我叫杨锜，来自中南大学计算机科学专业，大三学生。之前做过iOS独立开发，发布过FitGenius健身App，对AI和视频产品非常感兴趣。很高兴能加入芒果TV，希望在接下来的实习中能跟大家多学习、多贡献。"

## 七、入职第一天小贴士

```
✅ 着装: 商务休闲即可
✅ 提前10分钟到
✅ 带上笔记本和笔
✅ 准备一段30秒的自我介绍
✅ 心态: 空杯心态，多听多看多学
✅ 第一天不要急着表现，先融入
```

## 八、谈判补贴话术（可选）

> "HR您好，我已经决定入职了，非常期待加入团队。但考虑到我在长沙实习的租房和生活成本（约1500-2000元/月），能否申请每月500-1000元的实习补贴？我相信这份投资会让团队留住一个更用心投入的实习生。"

**谈判筹码**：
- 中南大学计算机科班
- 有iOS独立开发经验（FitGenius）
