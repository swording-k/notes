# 芒果TV 实习期发展规划

> 3个月的成长计划

## 整体节奏

```
第1月: 快速学习期
第2月: 实践产出期
第3月: 价值证明期
```

## 第1月：快速学习期

### 目标
```
- 熟悉业务 + 建关系
- 参与1-2个需求评审
- 开始承担小需求
- 独立完成1个完整小需求
```

### 周计划

**第1周**:
- 熟悉团队工作方式
- 主动约1对1咖啡
- 加入团队群，了解日常工作流
- 学习视频行业基础知识

**第2周**:
- 参与需求评审（旁听+记笔记）
- 学习产品文档模板
- 了解竞品（爱奇艺、腾讯视频、B站）

**第3周**:
- 开始承担小需求
- 写PRD、画原型
- 跨部门协作（设计/研发）

**第4周**:
- 独立完成1个完整小需求
- 月度复盘：学到了什么？

## 第2月：实践产出期

### 目标
```
- 独立负责1-2个完整功能
- 跨部门协作能力
- 数据分析能力
- 行业认知深化
```

### 学习重点

**数据分析**:
- 友盟 / 神策 / GA
- 用户行为分析
- 漏斗分析
- A/B测试

**产品方法论**:
- 用户调研
- 需求分析
- 功能设计
- 数据驱动

**视频行业**:
- 内容推荐算法
- 用户增长策略
- 会员订阅体系
- 创作者生态

## 第3月：价值证明期

### 目标
```
- 1-2个有数据成果的项目
- 拿到leader推荐信
- 准备转正答辩（如果走青芒生）
- 投递秋招
```

### 复盘清单

**应该收获的成果**:
- [ ] 1-2个完整的产品项目经验
- [ ] 1-2份产品文档（PRD、竞品分析）
- [ ] 团队内部的认可（leader推荐信）
- [ ] 视频行业的人脉
- [ ] 对视频行业产品方法的深度理解
- [ ] 1-2个数据成果（可以写进简历）

## 必学工具

```
✅ Axure / Figma（画原型）
✅ 语雀 / 飞书 / 钉钉（看文档）
✅ 禅道 / Jira / TAPD（看需求）
✅ 友盟 / 神策 / GA（看数据）
✅ ProcessOn / XMind（画思维导图）
```
