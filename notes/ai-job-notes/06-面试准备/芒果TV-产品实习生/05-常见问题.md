# 芒果TV 常见问题

## Q1: 芒果TV实习生无薪资，是行业普遍现象吗？

**A: 是的**。芒果TV/湖南广电的实习生制度有一个传统：
- 绝大多数岗位（运营/内容/设计）**无工资**
- 通常包午餐晚餐、咖啡半价
- 技术岗（开发/算法）通常有日薪100-300元

这不是你的谈价能力问题，是行业普遍现象。

## Q2: 无薪资实习值得去吗？

**A: 取决于你的目标**：

| 你的目标 | 建议 |
|---------|------|
| 想做视频/AI产品经理 | ✅ 强烈推荐去 |
| 想进大厂（通用） | ❌ 建议选有薪资的 |
| 想体验媒体行业 | ✅ 可以去 |
| 经济压力大 | ❌ 建议拒绝 |

## Q3: 如何判断是不是青芒生通道？

**问HR的关键问题**：
- "这个岗位是青芒生通道吗？"
- "表现优秀可以转正吗？"

**青芒生特点**：
- 芒果TV的正式校招实习项目
- 有正式薪资
- 表现优秀可转正
- 竞争激烈

## Q4: 实习期间如何再争取补贴？

**话术参考**：
> "HR您好，我已经决定入职了，非常期待加入团队。但考虑到我在长沙实习的租房和生活成本（约1500-2000元/月），能否申请每月500-1000元的实习补贴？"

**谈判筹码**：
- 中南大学计算机科班
- 有iOS独立开发经验（FitGenius）
- 懂AI技术
- 决定长期投入视频内容赛道

## Q5: 实习多久合适？

**建议**：
- 最短 2个月
- 最长 3-4个月
- 5个月以上容易变成"打杂"

## Q6: 如何在实习期间准备秋招？

**双线作战策略**：
```
工作日: 80%精力在芒果TV实习
晚上+周末: 20%精力投秋招、做笔试题、准备面试
```

**秋招时间线**：
- 7月：提前批（字节、快手、小红书、B站）
- 8月：正式批 + 芒果TV转正答辩
- 9月：补录批

## Q7: 这份实习对秋招有什么帮助？

**最有价值的产出**：
1. 视频行业经验（垂直背景）
2. 大厂背书（Top级视频平台）
3. AI+视频复合背景（极稀缺）
4. 团队推荐信
5. 数据成果（可以量化）

**简历写法**：
```markdown
芒果TV - 产品实习生（2025.06 - 2025.09）
- 负责XX业务线的需求调研和功能设计
- 参与XX功能从0到1的设计与上线
- 上线后用户留存提升XX%
- 深入了解视频内容生态
```

## Q8: 没有薪资的实习，对计算机科班学生性价比低吗？

**A: 是的，但对视频/AI方向不亏**：

**传统观点**（不推荐）：
- 计算机科班学生应该去字节/腾讯/阿里
- 这些公司有薪资、福利好、培训体系完整

**重新评估**（视频/AI方向）：
- 芒果TV是垂直行业大厂
- 视频行业经验稀缺
- 你的AI背景 + 视频经验 = 极稀缺

## Q9: 实习结束后怎么争取转正？

**关键点**：
1. 提前问清楚是否走青芒生通道
2. 拿到有数据成果的项目
3. 主动和leader沟通转正意向
4. 准备转正答辩（讲清楚贡献）
5. 拿到leader推荐信

## Q10: 如果实习完不想留芒果TV，怎么利用这段经历？

**秋招打法**：
1. 把芒果TV作为简历背书
2. 主投字节/快手/小红书/B站
3. 用"AI+视频"的复合背景差异化
4. 准备AI视频产品的深度思考
