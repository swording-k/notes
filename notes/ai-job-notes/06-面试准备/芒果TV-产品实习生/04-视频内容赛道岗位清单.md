# 视频内容赛道岗位清单

> 视频内容方向的产品岗位可以分为5大类

## 整体分类

```
视频内容产品岗位
├── 1. 内容平台型（小红书、抖音、B站）
├── 2. 视频工具型（剪映、CapCut）
├── 3. AI视频型（Sora、Runway、可灵）
├── 4. 视频社区型（得物、虎扑）
└── 5. 内容创作工具型（剪映、必剪）
```

## 闷声发大财的方向

### 1. 内容审核产品经理
```
需求: AI内容审核、平台治理
公司: 所有内容平台都需要
薪资: 高（涉及合规）
前景: 稳
```

### 2. 创作者生态产品经理
```
需求: UP主/博主工具、变现产品
公司: B站、小红书、抖音、YouTube
核心: 服务创作者
前景: 好（创作者经济是大趋势）
```

### 3. 视频商业化产品经理
```
需求: 广告系统、带货产品
公司: 抖音电商、小红书电商、视频号
核心: 流量变现
前景: 好（变现能力强）
```

### 4. 视频出海产品经理
```
需求: 海外市场拓展
公司: CapCut、Snack Video（字节海外）
核心: 国际化
前景: 非常好（海外市场大）
```

## 你的稀缺背景分析

### 简历关键词组合
```
芒果TV产品实习 + 计算机科班 + AI技术背景 + iOS独立开发者
```

### 背景维度
| 维度 | 价值 |
|------|------|
| 视频行业经验 | 懂视频业务的特殊性 |
| 大厂背景 | 芒果TV是国内Top级视频平台 |
| 计算机科班 | 能和技术团队深度对话 |
| AI技术背景 | 懂大模型、AI Agent |
| 独立开发者 | 有产品sense，懂用户 |
