## 全栈开发经验准备

### FitGenius 技术栈回顾

```
前端：SwiftUI（iOS原生）
后端：大模型 API（阿里云）
数据存储：本地存储
AI能力：RAG + 多模态（照片识别）
```

### 可能会被问到的问题

**Q: 为什么选择 SwiftUI 而不是 Flutter/React Native？**
A: 当时想深入学习 iOS 生态，SwiftUI 的声明式语法很适合 AI 辅助开发，代码生成质量高。

**Q: 大模型 API 集成遇到了什么问题？**
A: 主要挑战是 prompt 调优和 token 成本控制。通过设计系统 prompt 和缓存策略优化。

**Q: 项目的性能瓶颈在哪里？**
A: 大模型 API 响应时间。解决方案是添加 loading 状态和本地缓存。
