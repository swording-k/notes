```
美团
├── 到家事业群          # 外卖核心
│   ├── 餐饮外卖        # 美团核心业务，日单量7800万
│   ├── 美团闪购        # 即时零售，30分钟万物到家
│   ├── 美团配送        # 物流基础设施
│   ├── 美团买药        # 药品即时配送
│   └── 美团无人机       # 配送新方式
│
├── 到店事业群          # 利润最高
│   ├── 到店餐饮        # 团购
│   ├── 酒店旅游        # 酒店预订
│   └── 休闲娱乐        # 美容/电影等
│
└── 新业务             # 长期增长潜力
    ├── 美团买菜        # 生鲜电商
    ├── 美团打车        # 出行
    └── 零售相关
```

