## 🗺️ 学习路线

```
阶段一：Python基础（1周）
  └── 目标：会用Python写算法，不被语法卡住

阶段二：算法基础（1-2周）
  └── 数组 → 字符串 → 哈希表 → 栈队列 → 二分查找

阶段三：算法进阶（2-3周）
  └── 链表 → 递归回溯 → 贪心 → 动态规划

阶段四：AI专项（1-2周）
  └── 大模型基础 → RAG → Agent → LangChain

阶段五：机器学习基础（1周）
  └── 线性回归 → 梯度下降 → 归一化

阶段六：刷真题 + 冲刺（1-2周）
  └── 华为机考 + 目标公司真题
```

## 📝 笔记规范

**重要：所有笔记必须使用 Obsidian 内部链接格式**

```markdown
# 正确格式
[[笔记名]]                    # 链接同目录笔记
[[文件夹/笔记名]]              # 链接子目录笔记
[[../上级目录/笔记名]]         # 链接上级目录笔记
![[截图名.png]]               # 嵌入图片

# 错误格式（禁止使用）
```
