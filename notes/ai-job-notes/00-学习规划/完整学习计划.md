## 阶段四：机器学习基础（1周）

### 目标
能独立实现梯度下降，理解归一化原理

### 学习内容

#### 1. 线性回归（Day46-48）
```
核心概念：
- 线性回归模型：y = w0 + w1*x1 + w2*x2 + ...
- 损失函数：MSE（均方误差）
- 梯度下降：w = w - α * 梯度

必会：
- 手动实现梯度下降
- 理解归一化/标准化
- 理解学习率的影响
```

#### 2. 梯度下降详解（Day49-50）
```
核心理解：
- 批量梯度下降（BGD）
- 随机梯度下降（SGD）
- 小批量梯度下降

面试能说清楚：
- 什么是梯度？
- 为什么沿梯度反方向更新？
- 学习率太大/太小会怎样？
```

#### 3. 归一化与反归一化（Day51）
```
核心理解：
- Min-Max归一化：(x - min) / (max - min)
- 为什么要归一化？（加速收敛）
- 如何反归一化权重？（这是华为机考的重点）
```
