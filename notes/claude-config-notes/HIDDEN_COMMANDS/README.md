# 隐藏命令技巧

> 来源：10个你可能不知道的Claude Code隐藏命令

---

## 1. /btw (插入问题不污染上下文)

**功能**：在 Claude 正在工作时插入问题，但不会加入对话历史，不污染上下文。

**用法**：
```
/btw  [你的问题]
```
按空格或回车可以消除这段问答，不留痕迹。

---

## 2. /rewind (代码/对话分别回退)

**功能**：撤销代码或对话修改，支持选择性回退。

**选项**：
- 回退代码和对话
- 只回退对话，保留代码
- 只回退代码，保留对话
- 从该点压缩对话，释放上下文空间

**快捷键**：连续按两下 `Esc`

---

## 3. /insights (使用习惯分析)

**功能**：生成 HTML 报告，分析过去一个月的使用习惯，推荐自定义命令和 Skills。

**用法**：`/insights`

报告内容包括：
- 常用命令统计
- 重复操作模式
- 改进建议

---

## 4. /model opusplan (Opus 规划 + Sonnet 执行)

**功能**：复杂推理时自动使用 Opus 4.6，执行时切换 Sonnet 4.6，节省 Opus 额度。

**适用**：20刀的 Pro 订阅用户

---

## 5. /simplify (三合一代码审查)

**功能**：同时启动三个并行 Agent，从代码复用、代码质量、运行效率三个角度审查改动。

**用法**：`/simplify`

比 `/review` 更强大，相当于三个同事同时帮你 review。

---

## 6. /branch (对话分叉)

**功能**：将当前对话分叉出一个新会话，保留原会话不变。

**用法**：`/branch`

适合想试两个方向但不想丢掉当前进度时。

---

## 7. /loop (定时循环任务)

**功能**：让 Claude 定时重复执行某个任务。

**用法**：
```
/loop 5m 检查部署状态
```
默认间隔10分钟，3天后自动过期删除。

---

## 8. /remote-control (/rc) (手机遥控) ⚠️ 需桌面版

**功能**：生成 URL，手机打开后可远程控制 Claude Code。

**注意**：此功能需要 **Claude Code 桌面版**，终端版不支持。

---

## 9. /export (导出对话)

**功能**：将当前对话导出为 Markdown 文件。

**用法**：`/export`

适合保存架构讨论、方案推敲等重要对话。

---

## 10. 实用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+V` | 直接粘贴截图（Mac 用户） |
| `Ctrl+J` / `Option+回车` | 换行 |
| `Ctrl+R` | 搜索 prompt 历史 |
| `Ctrl+U` | 删除整行输入 |
| `Esc` × 2 | 回退代码/对话 |
